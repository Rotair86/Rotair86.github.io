eEmailForPurchaseOrder Enumeration

Types of auto-generated e-mail based on purchase orders.

Members
  Member name Description
None Default if no value is assigned.

ExistingPOWithForeignVendorAddedDefenseArticle Defense article(s) were added to the purchase order.
NewPOWithForeignVendor A purchase order with a foreign vendor has been created.
NewPOWithForeignVendorAndDefenseArticle A purchase order with a foreign vendor and defense article(s) has been created.
