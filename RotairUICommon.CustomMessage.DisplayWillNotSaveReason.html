DisplayWillNotSaveReason Method

Displays a messagebox indicating the reason, based on the screening return code, why an AppObject can not be saved.

Syntax
VB 
'Declaration
Public Sub DisplayScreeningFailure( _
        eWhich As eScreeningRC _
)

Parameters

eWhich
Type: eScreeningRC
Return code from the screening.
Remarks

This function will display a message concerning the screening status and why the AppObject can not be saved.

Example

If the party did not pass the screening test then the reason will be displayed when the user tries to save the AppObject.

VB 
If meScreenStatus <> Enumerations.eScreeningRC.Screened_OK Then
    CustomMessage.DisplayWillNotSaveReason(meScreenStatus)
End If
