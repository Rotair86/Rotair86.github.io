PersonName Constructor

Creates a PersonName object setting the name variables (first and last).

Syntax
VB 
'Declaration
Public Sub New( _
        strName As String _
)
Public Sub New( _
        strFirst As String, _
        strLast As String _
)
Public Sub New( _
        chrDelimiter As Char, _
        strName As String _
)

Parameters

strName
Type: System.String
Full name.
strFirst
Type: System.String
First name.
strLast
Type: System.String
Last name.
chrDelimiter
Type: System.Char
Character separating the first name from the last.
Remarks

Use the first option if the first and last names are separated by a space ("John Maher").

Use the second option if the first and last names are already separated.

Use the third option if the first and last names are separated by anything other than a space ("John-Maher").

Example

This example shows all three constructor calls.

VB 

Dim ThisName As PersonName

ThisName = New PersonName("John Maher")
ThisName = New PersonName("John", "Maher")
ThisName = New PersonName("-", "John-Maher")
