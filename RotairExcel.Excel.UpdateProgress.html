UpdateProgress Event

Allows use of a progress bar.

Syntax
VB 
'Declaration
Public Event UpdateProgress( _
        intCurrentRecord As Integer _
)

Parameters

intCurrentRecord
Type: Integer
Location of the data pointer.
Remarks

Fires with the current record count. The frequency of firing is predetermined in the class rotExcel.

Example

This example updates a progress bar when the event fires. The max is set to the total record count.

VB 
Private Sub XL_UpdateProgress(ByVal intCurrentRecord As Integer) Handles XL.UpdateProgress

    ProgressBar.Value = intCurrentRecord

End Sub
