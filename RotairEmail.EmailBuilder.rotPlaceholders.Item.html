Item Method

Returns the value of a token.

Syntax
VB 
'Declaration
Public Function Item( _
        strName As String _
) As String

Parameters

strName
Type: System.String
The token whose cooresponding value we wish to retrieve.

Return Value

Type: System.String
The value associated with the passed in token.
Remarks

Pass this function a name of a token and it returns the value. If the token was not set then will return an error string. For example, if a token called Name has its value requested and it has not been set then the string "!!!Error:!!! Placeholder 'Name' is undefined." will be returned.

Example

The example will try to retrieve the value for the token "LogInUserName" and display the result in a message box. We assume a PlaceHolders collection called mcolPH has been instantiated.

VB 
Private Sub Main()

    RetrieveTokensValue("LogInUserName")

End Sub
