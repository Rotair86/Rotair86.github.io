PopulateData Method

Retrieves the data from the database.

Syntax
VB 
'Declaration
Public Sub PopulateData( _
)
Remarks

A table as far as reports go is a block of data which can come from a SQL statement pulling data from one or more database tables. A report can contain more than one report table (SQL statements). A report table exists for each row in the tblReportTable table. This function will populate a datatable in the dataset for each report table (row in tblReportTable).

Example

This example displays a report to give the user the option to print. The actual report uses a guid instead of a string id. The string is displayed here for clarity. Also with this particular report a couple of fields need to be set. The example shows the minimum that needs to be done.

VB 
Dim Report As rotReport

Report = New rotReport("ScreeningCustomerOrder", "623454")
With Report
    .PopulateData()
    .UpdateReportHeader()
    .CopyDataToReporter()
    .PrintPreview()
End With
