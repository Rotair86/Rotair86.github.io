RFQIndex Method

Returns true if the logged in employee has permission to index RFQs.

Syntax
VB 
'Declaration
Public Function RFQIndex( _
) As Boolean

Return Value

Type: System.Boolean
TRUE if the employee can index RFQs.
Remarks

Returns TRUE if the logged in user is a member of the RFQ index or system administrator security group.

Example

The example displays a messagebox if the current user can index RFQs.

VB 

If  Rotair_Security.Security.RFQIndex() Then
    MessageBox.Show("You can index RFQs.")
End If
