Notes

To use an imsForm as a dialog an imsAppObject must be used to pass data back and forth. These forms should eventually be changed to imsForms. To display an imsForm as a dialog you must use the imsLaunchProcessCommand object.

All events should have basic error trapping. Below is an example of the minimum recommended. It uses Intuitive's form to display errors and also writes out the error to the database including the class and function where the error was trapped.

In order to use Intuitive's error form in a windows form an imsForm must be declared. This is not necessary when using an imsForm because the function is intrinsic to the form.

VB 
Try

    'Code goes here.

Catch ex As Exception
    Dim ErrorForm As New imsForm
    ErrorForm.HandleException(ex)
    RotairDataAccess_Wrapper.Database.WriteToErrorTable(ex.TargetSite.ReflectedType.Name, _
                ex.TargetSite.Name, ex.Message)

End Try
