IsUseClassDetermined Method

Indicates if the use class was specified for an item.

Shared function because we do not need an object for the results and also to self document.

Syntax
VB 
'Declaration
Public Shared Function IsUseClassDetermined( _
        strItemID As String _
    ) As Boolean

Parameters

strItemID
Type: System.String
The item id.

Return Value

Type: System.Boolean
TRUE if a use class was selected.
Remarks

Returns TRUE if a user selected a use class then saved the item. If no use class was specified then the use class is set to "Undetermined" and the function returns FALSE.

Example

The example will display a message if the use class was not set for the passed in item.

VB 
Dim bRC As Boolean


bRC = imsAO_UseClass.IsUseClassDetermined("Widget")
If Not bRC Then
    MessageBox.Show("Must choose the use class of the item.")
End If
