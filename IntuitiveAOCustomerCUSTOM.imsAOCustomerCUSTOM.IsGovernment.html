IsGovernment Method

Returns TRUE for the governemnt.

Syntax
VB 
'Declaration
Public Function IsGovernment(Of CustomerKey)( _
        CustomerID As CustomerKey _
) As Boolean

Type Parameters

CustomerKey
The type of customer key used. Can be a string or guid.

Parameters

CustomerID
Type: CustomerKey
Key for the customer.

Return Value

Type: System.Boolean
TRUE if the customer is the government.
Remarks

Returns TRUE if the customer type begins with "US Gover".

Example

The example determines if the customer is the government on the shipment card.

VB 
Dim strCustomerID As String
Dim bIsGovernment As Boolean
Dim strSalesOrderID As String

strSalesOrderID = KeyInputSO_fndSalesOrderFind.SelectedID
strCustomerID = msAO_Customer_CUSTOM.IDBySalesOrderID(strSalesOrderID)
bIsGovernment = msAO_Customer_CUSTOM.IsGovernment(strCustomerID)
