rotExcel Constructor

Initializes a new instance of rotExcel passing in a file path of a document.

Syntax
VB 
'Declaration
Public Sub New( _
        strFilePath As String _
)

Parameters

strFilePath
Type: String
Path for the file being created or opened.
Remarks

Instantiating a new rotExcel object requires a file path.

Example

This example creates a new rotExcel object then passes it a datasource, saves the file and finally closes all opened workbooks.

VB 
Dim XL As rotExcel

XL = New rotExcel("C:\My Documents\Spreadsheet.xls")
XL.ShowData(Enumerations.eExcelReport.Booking, Grid.DataSource)
XL.SaveFile()
XL.Dispose()
