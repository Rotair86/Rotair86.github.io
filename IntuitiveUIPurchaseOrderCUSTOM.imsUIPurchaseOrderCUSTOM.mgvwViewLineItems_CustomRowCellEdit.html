mgvwViewLineItems.CustomRowCellEdit Event

Prohibits entering of undetermined items.

Syntax
VB 
'Declaration
Private Event CustomRowCellEdit As EventHandler
Remarks

Gets a reference to the control used for editing to trap the validating event so we can stop undetermined items being entered into the grid. Also removes the button used in the item lookup for the same reason, no easy way to filter the results.

Example

Gets the reference for the editing control and removes the item lookup button.

VB 
Private Sub mgvwViewLineItems_CustomRowCellEdit( _ 
    ByVal semder As System.Object, _
    ByVal e As DevExpress.XtraGrid.Views.Grid.CustomRowCellEditEventArgs _
    ) Handles mgvwViewLineItems.CustomRowCellEdit

    If e.Column.Name = "gvwItemLines_collNoRevItemID"  Then
        'Get a reference to the control used for entering the item id.
        mribeItemID = CType(e.RepositoryItem, DevExpress.XtraEditors.Repository.RepositoryItemButtonEdit)
        'Remove the item lookup button; actually removes all the buttons of which there is only one.
        mribeItemID.Buttons.Clear()
    End If

End Sub
