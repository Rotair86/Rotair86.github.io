eLoadResultType Enumeration

Return code from a rotair object's load function.

Members
  Member name Description
UnknownError If an exception is thrown then this is returned.
Success Data was successfully loaded.
SuccessLocked Data was successfully loaded and a lock is on the record.
SuccessReadOnly Data was successfully loaded and is in a read only state.
NoData No records were returned.
NotImplemented The load function does not work with this object.
