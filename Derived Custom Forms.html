Derived Custom Classes

Derived Custom Classes refers to classes that are derived from a custom class.

Our namespaces are analogous with our DLLs in that there is one logical task per. The DLLs are simple enough not to warrant an additional namespace. Folowing is a list of all the DLLs grouped by type in alphabetical order. Naming conventions used: modified form of hungarian notation.
  • AO: A namespace representing data in the database.
  • UI: A MDI form inheriting from imsForm.
  • Dialog: A windows form displayed as a dialog .
  • HC: A namespace that handles printing (hard copy).

Helper Namespaces

The only thing in common with these namespaces is that they do not have anything in common.

Name Description
Rotair_Types Constants, enumerations and structures that need to be shared accross namespaces.
Rotair_GeneralFunctions Function repository.
RotairDataAccess The data access layer. The only namespaces that should reference this are the application objects; namespaces prefixed with AO and the wrapper. This is the only namespace that should interact with SQL server.
RotairDataAccess_Wrapper Used as a bridge between non-AO namespaces and the data access layer. Ideally these functions should be move to their respective AOs and this namespace eliminated.
Rotair_Security Checking of menu item permissions lives here.
Rotair_Email This namespace is used to send out emails.
Rotair_ExportManagement Mainly used for screening.
Rotair_Report Prints out reports using data from RotairAO_Report and sending it to RotairHC_Report.
Rotair_Excel Talks to Excel.
RotairUI_Common Messageboxes and forms used in more than one place.

Notes

All events should have basic error trapping. Below is an example of the minimum recommended. It uses Intuitive's form to display errors and also writes out the error to the database including the class and function where the error was trapped.

VB 
Try

    'Code goes here.

Catch ex As Exception
    HandleException(ex)
    RotairDataAccess_Wrapper.Database.WriteToErrorTable(ex.TargetSite.ReflectedType.Name, _
                ex.TargetSite.Name, ex.Message)

End Try
