SendPurchaseOrderEMail Method

Sends various e-mails including mails relating to defense articles and foriegn vendors.

Syntax
VB 
'Declaration
Public Shared Function SendPurchaseOrderEMail( _
        eEmailType As eEmailForPurchaseOrder, _
        guidVendorRecordID As Guid, _
        strPurchaseOrderID As String _
) As Boolean

Parameters

eEmailType
Type: RotairTypes.Enumerations.eEmailForPurchaseOrder
The specific type of purchase order mail.
guidVendorRecordID
Type: System.Guid
The vendor record id associated with this purchase order.
strPurchaseOrderID
Type: System.String
The purchase order id.

Return Value

Type: System.Boolean
TRUE if no problems were found.
Remarks

Sets all the placeholders needed and determines the e-mail type then calls the SendClassBasedEmail function.

Example

This function will get passed a PurchaseOrder object and pass it to a function that will return TRUE if the purchase order contains any defense articles. If any items are found an e-mail is sent.

VB 

Private Sub CheckForDefenseArticles(ByVal PurchaseOrder As imsAO_PurchaseOrder)

    'Does the purchase order contains a defense article?
    If HasDefenseArticle(PurchaseOrder) Then
        'Send an e-mail.
        SendPurchaseOrderEMail(Enumerations.eEmailForPurchaseOrder.NewPOWithForeignVendorAndDefenseArticle, _
            SalesOrder.GetString("POM_VEN_RecordID"), _
            SalesOrder.GetString("POM_PurchOrderID"))
    End If

End Sub
