ClearStatus Method

Removes controls added to an imsPanel.

Syntax
VB 
'Declaration
Public Sub ClearStatus( _
        pnl As imsPanel, _
        eWhich As eShowStatus _
)

Parameters

pnl
Type: imsPanel
The panel the controls were added to.
eWhich
Type: eShowStatus
Indicates which type of control we are clearing.
Remarks

Removes both labels and buttons, whichever is present.

Example

The example clears the footer panel when the form is reset.

VB 
Protected Overrides Sub ResetForm()

    MyBase.ResetForm()
    UI.ClearStatus(pnlFooter, UI.eShowStatus.All)

End Sub
