CurrentUserName Method

Returns the logged in employee's name.

Syntax
VB 
'Declaration
Public Function CurrentUserName( _
        eNameStyle As eNameReturnStyle, _
) As String

Parameters

eNameStyle
Type: eNameReturnStyle
Determines the format that the name is returned.

Return Value

Type: System.String
The employee's name.
Remarks

Returns the name of the employee who is logged in to the ERP system which can be different to the user who is logged into the computer.

Example

The example will retrieve the name of the logged in user in two different styles which is found in the employee table.

VB 
Dim strName As String


strName = imsAO_Employee_CUSTOM.CurrentUserName(eNameReturnStyle.LastFirst)
strName = imsAO_Employee_CUSTOM.CurrentUserName(eNameReturnStyle.IDFirstLast)
