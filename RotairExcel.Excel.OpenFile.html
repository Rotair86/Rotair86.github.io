OpenFile Method

Opens a file in Excel.

Syntax
VB 
'Declaration
Public Sub OpenFile( _
)
Remarks

This will open a file into Excel. The file and path are passed into the object during instantiating.

Example

This example opens an Excel file.

VB 
Dim DS As DataSet

XL = New rotExcel("C:\My Documents\Spreadsheet.xls")
XL.OpenFile()
