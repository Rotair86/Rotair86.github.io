ScreenEndUserCheckExportLicense Method

Checks an end user's screening.

Syntax
VB 
'Declarations
Public Function ScreenEndUserCheckExportLicense( _
        strSalesOrderID As String, _
        strEndUser As String, _
        strExLicenseID As String _
) As eScreeningRC

Parameters

strSalesOrderID
Type: System.String
Sales Order ID for this end user.
strEndUser
Type: System.String
End User as entered on the Sales Order.
strExLicenseID
Type: System.String
Export License ID.

Return Value

Type: eScreeningRC
Results of the screening.
Remarks

Returns the results of an end user screening.

Example

The example will display a message depending on the results of an end user screening.

VB 
strEndUser As String
strExLicenseID As String
eRC As Enumerations.eScreeningRC

eRC = ExportManagement.ScreenEndUserCheckExportLicense( _
        "623454", _
        strEndUser, _
        strExLicenseID)
Select Case eRC
    Case Enumerations.eScreeningRC.End_User_Never_Screened
        Messagebox.Show("End user needs to be screened.")
    Case Enumerations.eScreeningRC.End_User_Screening_Out_of_Date 
        Messagebox.Show("End user needs a current screening.")
    Case Enumerations.eScreeningRC.Export_License_Expired
        Messagebox.Show("Need to apply for a new export license.")
    Case Enumerations.eScreeningRC.Export_License_Not_Approved
        Messagebox.Show("Waiting for the export license to be approved.")
    Case Enumerations.eScreeningRC.Screened_OK
        Messagebox.Show("Ready to ship.")
End Select
