ID Method

Returns a string id from a record id (guid).

Syntax
VB 
'Declaration
Public Function ID( _
        guidCustomerID As Guid _
) As String

Parameters

guidCustomerID
Type: System.Guid
The customer record id.

Return Value

Type: System.String
The customer string id.
Remarks

Returns a string id based on a record id.

Example

The example retrieves the customer id using the record id on the quote card.

VB 

Dim strCustomerID As String

strCustomerID = imsAO_Customer_CUSTOM.ID(bdrQuote.AO.GetGuid("QH_CUS_RecordID"))
