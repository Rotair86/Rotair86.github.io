ScreeningAllowAdvanced Method

Returns true if the logged in employee has permission for advanced screening functions.

Syntax
VB 
'Declaration
Public Function ScreeningAllowAdvanced( _
) As Boolean

Return Value

Type: System.Boolean
TRUE if the employee can access advanced screening functions.
Remarks

Returns TRUE if the logged in user is a member of the export admin or system administrator security group.

Example

The example displays a messagebox if the current user can access advanced screening functions.

If  Rotair_Security.Security.ScreeningAllowAdvanced() Then
    MessageBox.Show("You can undeny people.")
End If
