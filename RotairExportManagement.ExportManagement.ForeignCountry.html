ForeignCountry Method

Checks a country name.

Syntax
VB 
'Declarations
Public Function ForeignCountry( _
        strCountry As String _
) As Boolean

Parameters

strCountry
Type: System.String
Country to check.

Return Value

Type: System.Boolean
TRUE if the country is a foriegn country.
Remarks

Returns TRUE if the passed in country is a foriegn country. The country is validated against "US", "USA", "United States of America" or any other combination deemed necessary to check against. A null string will validate as FALSE.

Example

The example will display a message if the country passes as a forieng country.

VB 
If ExportManagement.ForeignCountry("Iran") Then
    Messagebox.Show("Possible foriegn shipment.")
End If
