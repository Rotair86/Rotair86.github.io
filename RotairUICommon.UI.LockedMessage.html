LockedMessage Method

Displays a message.

Syntax
VB 
'Declaration
Public Sub LockedMessage( _
        strUser As String, _
        strMachine As String _
)

Parameters

strUser
Type: System.String
The user holding the lock.
strMachine
Type: System.String
The machine the lock was created on.
Remarks

Displays a message when a record that a user is trying to access is locked by another user.

Example

This example will display a message if a CageCode is loaded that is locked by another user. The CageCode is really loaded by a guid but a string is used here for clarity.

VB 
Dim mCageCode As rotAO_CageCode
Dim eLoadOK As eLoadResultType

    mCageCode = New rotAO_CageCode("378923")
    eLoadOK = mCageCode.Load()
    If eLoadOK = Enumerations.eLoadResultType.SuccessLocked Then
        RotairUI_Common.UI.LockedMessage(mCageCode.LockedBy, mCageCode.LockedOnHost)
    End If

End Sub
