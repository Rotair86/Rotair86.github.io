GetDate Method

Returns a date or nothing.

Syntax
VB 
'Declarations
Public Function GetDate( _
        Reader As System.Data.SqlClient.SqlDataReader, _
        intColumn As Integer _
) As DateTime?

Parameters

Reader
Type: SQLDataReader
A reader with a date column.
intColumn
Type: System.Integer
The ordinal of the date column.

Return Value

Type: DateTime?
A valid datetime or nothing.
Remarks

Returns a datetime if there is valid data at the passed in column otherwise returns nothing.

Example

The example will display a message indicating if the birthdate is found or not.

VB 
Dim dtmBirthDate As DateTime?
Dim cmd As imsCommand
Dim Reader As SqlDataReader


cmd = New imsCommand
cmd.CommandText = "SELECT EMP_BirthDate FROM EMP WHERE EMP_EmployeeID = '943'"

Using DataServices As NewimsDataServices
    Reader = DataServices.OpenDataReader(cmd)
    dtmBirthDate = Rotair_GeneralFunctions.GetDate(Reader, 0)
    If dtmBirthDate Is Not Nothing Then
        Messagebox.Show("DOB found")
    Else
        Messagebox.Show("DOB not found")
    End If
End Using
