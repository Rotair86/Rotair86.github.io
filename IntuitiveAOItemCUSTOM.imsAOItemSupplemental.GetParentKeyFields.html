GetParentKeyFields Function

Retrieves the parents primary key.

Syntax
VB 
'Declaration
Private Sub GetParentKeyFields()
Remarks

This function was created per the user documentation. It does not necessarilly need to be done this way.

Example

This function uses the variables mParentKey and mParentKeyFieldName to get the parents key and set this subdomains foriegn key to its value.

VB 
Private Sub GetParentKeyFields()

    'Ensure that the MyParent property returns a valid AppObject.
    If Not MyBase.MyParent Is Nothing OrElse Not TypeOf (MyBase.MyParent) Is imsAppObject Then
        mParentKey = MyParent.GetValue(mParentKeyFieldName)
    End If

End Function
