EmployeeVerification Method

Returns true if the passed in password matches the employee's password on file.

Syntax
VB 
'Declaration
Public Function EmployeeVerification( _
        strID As String, _
        strPassword As String _
) As Boolean

Parameters

strID
Type: System.String
The id for the employee to verify.
strPassword
Type: System.String
The password to check.

Return Value

Type: System.Boolean
TRUE if the passwords match.
Remarks

Returns TRUE if the supplied password matches the passord on file for the passed in employee id.

Example

The example displays a messagebox if the password does not match.

VB 

If Not Rotair_Security.Security.EmployeeVerification("943", txtPassword.Text) Then
    MessageBox.Show("Invalid Password")
End If
