aocCustomer.Saving Event

Fired before the custoemr AO is saved.

Syntax
VB 
'Declaration
Private Event Saving As EventHandler
Remarks

This event fires when Accept is clicked. It is used to prohibit saving of a customer record when they have not passed the screening test.

Example

This example prohibits saving and displays a message if the boolean mbOK_To_Save (modular level set elsewhere) is FALSE.

VB 
Private Sub aocCustomer_Saving( _ 
    ByVal sender As System.Object, _
    ByVal e As System.ComponentModel.CancelEventArgs _
    ) Handles aocCustomer.Saving

    If Not mbOK_To_Save Then
        MessageBox.Show("Unable to save!")
        e.Cancel = True
    End If

End Sub
