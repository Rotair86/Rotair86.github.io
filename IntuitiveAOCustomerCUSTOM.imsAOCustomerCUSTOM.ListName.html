ListName Function

Retrieves customer's names.

Public Function ListName( _
    ) As DataSet
Remarks

Returns a data set with customer's names and ids. A sample is provided below.

ID Name
000006 O/H - I.P.
000010 A-1 AIR & MARINE SUPPLY

Example

Populates an Intuitive combobox with the customer names. The combobox must already have its ListView set up for 2 columns and the display, key and value members set.

VB 
Private Sub PopulateCustomers( _
    )

    Dim DS As DataSet
    Dim DT As DataTable


    DS = imsAO_Customer_CUSTOM.ListName()
    DT = DS.Tables(0)
    ImsCombo1.ListView.DataSource = DT

End Sub
