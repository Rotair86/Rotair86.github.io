ClearLastError Method

Resets the error information.

Syntax
VB 
'Declaration
Public Shared Sub ClearLastError
Remarks

Works in conjunction with the LastError property.

All error information is reset, which includes and is limited to mstrLastError. It is recommended to call this function before calling any function where you wish to inspect LastError when the call returns.

Example

The example will call a function to send some sales order related e-mail. ClearLastError is called first to make sure no previous error information has been retained. Then it will call the function to send the e-mail, displaying any error information in a message box if there is any.

VB 
Rotair_Email.rotEmailBuilder.ClearLastError()
If Not Rotair_Email.rotEmailBuilder.SendSalesOrderRelated( _
            Enumerations.eEmailForSalesOrder.ExistingExportOrderWithOutSubmittedLicense, _
            strSalesOrderID, strCustomerID)
    MessageBox.Show("Mail may not have been sent." & Environment.NewLine & _
            Rotair_Email.rotEmailBuilder.LastError)
End If
