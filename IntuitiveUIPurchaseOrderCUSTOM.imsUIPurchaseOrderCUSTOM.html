Customizations

Informational Footer

The footer has been transitioned to a display of certain key information that is normally visible on a specific tab so that it is always visible.

A label gets displayed for the following:

  • DeniedParty is TRUE; displays "DP Stop".
  • Records exist in tblRI_Note; displays "Notes Exist".

Notes

The last notes entered for line items, if there are any, are displayed in the grid. Notes for the header are not displayed. To view them you need to go to the notes form.

Notes for the header can be added by clicking on the Notes link label in the task shade. For a line item clicking on the cell displays a button that will bring up the notes form.


Screening

A purchase order can only be edited if the vendor has a successful screening. This is determined by calling ScreenVendor. If the vendor fails to pass the screening the user is presented with the option to screen the vendor via the Screening dialog.


Phrases

Predefined blocks of text are stored in the database and added to the purchase order header or item lines via the Phrases form.


Undetermined Items

Undetermined items are not allowed on the purchase order. The item lookup button has been removed and the item id is checked when entered in the grid.
