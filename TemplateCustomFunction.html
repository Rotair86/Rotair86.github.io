Remarks

Displays the purchase order ao in a form by calling ShowAO. You can pass in a title that is used for the form's title. Useful if you are comparing objects in different functions.

Example

Displays the id with the purchase order, the line number and item id for the purchase order line item, and the quantity and date for the purchase order delivery. Fields to be displayed can be added, changed or deleted by modifying the array passed to the function.

VB 
Public Sub DisplayPurchaseOrder( _
    strTitle As String
    )

    RotairUI_Common.UI.ShowAO(strTitle, bdrPurchaseOrder.AO, New String() { _
            "1AlphaID-POM_PurchOrderID", _
            "2Line-POI_POLineNbr", _
            "2Item ID-POI_ItemID", _
            "3Quantity-POD_RequiredQty", _
            3})

End Sub
