frmDialogPhrases Form

Resizeable form used to modify a Phrases field.

This form uses a simulated combobox. It is really 2 listboxes and a button. When the button is clicked, the listbox will increase its height. The reason for 2 listboxes is there are two selection modes; single and multi-select. Single select uses a listbox. Multi-select uses a checkedlistbox.

To use this form the PhrasesSelected event must be handled. See Usage for instructions.
