Informational Footer

The footer has been transitioned to a display of certain key information that is either visible on a specific tab or not displayed at all.

A label gets displayed for the following:

  • DeniedParty is TRUE; displays "DP Stop".
  • Records exist in tblRI_Note; displays "Notes Exist".

Notes

The last note entered, if there is one, is displayed in the Last Note textbox.

Notes for a vendor can be added by clicking on the Notes link label in the task shade. They also can be entered by double-clicking on the Last Note textbox.


Screening

A vendor can only be edited if they have a successful screening. This is determined by calling ScreenVendor. If the vendor fails to pass the screening the user is presented with the option to screen the vendor via the Screening dialog.
