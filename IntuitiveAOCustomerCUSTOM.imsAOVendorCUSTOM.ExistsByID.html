ExistsByID Method

Returns TRUE if the vendor exists.

Syntax
VB 
'Declaration
Public Function ExistsByID(Of VendorKey)( _
        VendorID As VendorKey _
) As Boolean

Type Parameters

VendorKey
The type of vendor key used. Can be a string or guid.

Parameters

VendorID
Type: VendorKey
Key for the vendor.

Return Value

Type: System.Boolean
TRUE if the vendor exists.
Remarks

Returns TRUE if the vendor key exists in the database.

Example

The example will display a message if the vendor does not exist.

VB 
If imsAO_Vendor_CUSTOM.ExistsByID("176930") Then
    Messagebox.Show("Vendor does not exist")
End If
