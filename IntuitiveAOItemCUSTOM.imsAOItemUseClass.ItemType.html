ItemType Method

Returns the use class of an item.

Shared function because we do not need an object for the results and also to self document.

Syntax
VB 
'Declaration
Public Shared Function ItemType(Of ItemKey)( _
        ItemID As ItemKey _
    ) As eItemUseClass

Type Parameters

ItemKey
The type of item key used. Can be a string or guid.

Parameters

strItemID
Type: ItemKey
The item id.

Return Value

Type: eItemUseClass
The use class of the item.
Remarks

Returns a eItemUseClass enumeration representing the use class.

Example

The example will retrieve the use class for an item abd display a message indicating the results.

VB 
Dim eRC As eItemUseClass


eRC = imsAO_UseClass.ItemType("Widget")
Select Case eRC
    Case eItemUseClass.Commercial
        MessageBox.Show("Commercial item.")
    Case eItemUseClass.Defense_Article
        MessageBox.Show("Defense item.")
    Case eItemUseClass.Dual_Use
        MessageBox.Show("Commercial and defense item.")
    Case eItemUseClass.Undetermined
        MessageBox.Show("Unknown.")
End Select
