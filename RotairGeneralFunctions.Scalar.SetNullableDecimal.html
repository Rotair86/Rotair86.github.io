SetNullableDecimal Method

Returns a value or nothing for a null.

Syntax
VB 
'Declarations
Public Function SetNullableDecimal( _
        value As Object _
) As Decimal?

Parameters

value
Type: System.Object
A value from the database that can be nothing.

Return Value

Type: System.Decimal
The passed in value or nothing if no valid data was passed in.
Remarks

Pass this function an object and it will convert it to a decimal if possible. If not, then it will return nothing. Use it for nullable types that can expect nulls from the database.

Example

The example calls a function that will return a field from the database. This field can be null so passing it to the function will either convert the value (if valid data was returned) or assign nothing.

VB 
Dim SomeData As Object
Dim decNullableDecimal As Decimal?


'Let's get a possible decimal.  No data can be returned from the following function.
LoadData(SomeData)
'Now we assign to the nullable variable.
decNullableDecimal = Rotair_GeneralFunctions.SetNullableDecimal(SomeData)
