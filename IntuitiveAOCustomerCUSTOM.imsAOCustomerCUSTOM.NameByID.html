NameByID Method

Returns the customer's name based on the key.

Syntax
VB 
'Declaration
Public Function NameByID(Of CustomerKey)( _
        CustomerID As CustomerKey _
) As String

Type Parameters

CustomerKey
The type of customer key used. Can be a string or guid.

Parameters

CustomerID
Type: CustomerKey
Key for the customer.

Return Value

Type: System.String
The customer's name.
Remarks

Returns the customer's name.

Example

The example will get the customer name from the change / history tracking field of the UseClass object.

VB 
Dim strName As String

strName = imsAO_Employee_CUSTOM.NameByID(eNameReturnStyle.FirstLast, _
        UseClass.GetString("LastRevBy"))
