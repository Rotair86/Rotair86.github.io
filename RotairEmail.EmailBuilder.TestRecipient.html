TestRecipient Property

Turns off the recipient list in order to send to a specific recipient.

Syntax
VB 
'Declarations
Public Shared Property TestRecipient As Boolean

Remarks

If this property is not an empty string then the recipient list is not used and any messages go to the e-mail stored in this property.

Example

The example tests the SendScreeningEMail function.

VB 
'Set a recipient.
Rotair_Email.rotEmailBuilder.TestRecipient = "JohnM@rotair.com"
'Test the results.
SendScreeningEMail(Enumerations.ePartyType.Customer, _
            "000000", _
            Enumerations.eScreenAction.Open_A_Customer, _
            "TestForm")
'Clear the recipient.
Rotair_Email.rotEmailBuilder.TestRecipient = String.Empty
