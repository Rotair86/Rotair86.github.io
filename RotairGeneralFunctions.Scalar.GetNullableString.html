GetNullableSingle Method

Returns a value supplying a default for nothing.

Syntax
VB 
'Declarations
Public Function GetNullableString( _
        value As Object _
) As String

Parameters

value
Type: System.Object
An object that can be nothing or be holding a string.

Return Value

Type: System.String
The passed in string or nothing if nothing was passed in.
Remarks

Pass this function an object and it will return the string value if it is not nothing. If the passed in value is nothing then it will return nothing.

This function may not be necessary as a string derives from object and can be nothing intrinsically.

Example

The example will set a variable. But we may not have data, that is why the variable is defined as nullable. So we call the function that will assign a default value if nothing was assigned to the variable.

VB 
Dim objSomeData As Object
Dim strSomeString As String


'Let's load the object objSomeData.
LoadData(objSomeData)
'Now we need to convert objSomeData (which may or may not have been set) to strSomeString.
strSomeString = Rotair_GeneralFunctions.GetNullableString(objSomeData)
