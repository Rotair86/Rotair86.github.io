ToString Method

Returns the quote line number, itemid and the quoted quantity.

Syntax

Return Value

Type: System.String
Line number, itemid and quantity.
Remarks

Returns the line number followed by a colon then the itemid in parenthesis and lastly the quantity.

Example

The example is code taken from the sales order form. The form fills the combo when it is selected. The example looks to fill the combo once so may not run as expected. It is just to demonstrate that a structure (or class) in a list will have its ToString function automatically called for display purposes.
