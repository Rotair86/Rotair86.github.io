GetNullableSByte Method

Returns a value supplying a default for nothing.

Syntax
VB 
'Declarations
Public Function GetNullableSByte( _
        value As SByte? _
) As SByte

Parameters

value
Type: Nullable
A sbyte that can be nothing.

Return Value

Type: System.SByte
The passed in value or 0 if nothing was passed in.
Remarks

Pass this function a nullable sbyte and it will return the sbyte value if it is not nothing. If the passed in value is nothing then it will return 0.

Example

The example will set a variable. But we may not have data, that is why the variable is defined as nullable. So we call the function that will assign a default value if nothing was assigned to the variable.

VB 
Dim sbytSomeSByte As SByte?
Dim sbytAnotherSByte As SByte


'Let's load the byte sbytSomeSByte.
LoadData(sbytSomeSByte)
'Now we need to convert sbytSomeSByte (which may or may not have been set) to sbytAnotherSByte.
sbytAnotherSByte = Rotair_GeneralFunctions.GetNullableSByte(sbytSomeSByte)
