CheckSalesOrderExportLicense Method

Provides export license information.

Syntax
VB 
'Declarations
Public Function CheckSalesOrderExportLicense( _
        strSalesOrderID As String _
) As eSalesOrderExportLicense

Parameters

strSalesOrderID
Type: System.String
The sales order id.

Return Value

Type: eSalesOrderExportLicense
The status of the export license.
Remarks

Returns the status of an export license.

Example

The example will display a message with the contents depending on the status of the export license.

VB 
eExportLicenseStatus As Enumerations.eSalesOrderExportLicense

eExportLicenseStatus = ExportManagement.CheckSalesOrderExportLicense("623454")
Select Case eExportLicenseStatus
    Case Enumerations.eSalesOrderExportLicense.Customer_Supplied
        Messagebox.Show("Customer is supplying ther license.")
    Case Enumerations.eSalesOrderExportLicense.License_Submitted 
        Messagebox.Show("License has been subbmitted.")
    Case Enumerations.eSalesOrderExportLicense.No_License_Action
        Messagebox.Show("No license is required.")
End Select
