Value Property

Provides access to a value that replaces a token placeholder.

Syntax
VB 
'Declarations
Public Property Value As String

Remarks

This property returns the value which replaces a token, Name.

Example

The example will replace anyone's age that is less than 18 with the word "Minor".

VB 
Private Function HideMinorsAge() As Boolean

    Dim bFound As Boolean
    Dim PH As rotPlaceholder


    For Each PH In mcolPH
        If Val(PH.Value) < 18 Then
              PH.Value = "Minor"
        End If
    Next

End Function
