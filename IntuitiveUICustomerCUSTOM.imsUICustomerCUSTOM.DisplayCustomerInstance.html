DisplayCustomerInstance Function

Displays some custom information after a customer screening.

Syntax
VB 
'Declaration
Protected Overrides Sub DisplayCustomerInstance
Remarks

This function handles a number of things:.

  1. Screens the customer by calling GetScreeningInfo.
  2. If not cancelled completes the following:
    1. Displays a label in the footer if the customer is designated "No Quote" by calling ShowStatus.
    2. Calls the base function.
    3. Displays a label in the footer to indicate when the customer has notes by calling ShowStatus.
    4. Calls NoteCount and LastNote to display how many notes exist for this customer and the last note entered if there is one.
Example

The example just shows a portion of what this function does. This function doesn't get called directly.

VB 
Private Sub DisplayCustomerInstance()

    GetScreeningInfo()
    'If the user clicks cancel from then the customer ao will be nothing.
    If aocCustomer.AO IsNot Nothing Then
        'Display a 'NoQuote' label if the customer is designated as such.
        If CType(aocCustomer.AO.GetValue("RI_NoQuote"), Boolean) Then
            UI.ShowStatus(pnlFooter, UI.eShowStatus.NoQuote)
        End If

        MyBase.DisplayCustomerInstance()

        'Display a 'Notes' label if the customer has at least one note.
        If rotAO_NoteType.NoteCount("Customer", "BA404025-7781-4176-8367-1E342BA0BE6C") > 0 Then
            UI.ShowStatus(pnlFooter, UI.eShowStatus.NotesExist)
        End If

    End If

End Sub
