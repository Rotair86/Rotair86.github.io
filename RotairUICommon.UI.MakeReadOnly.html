MakeReadOnly Method

Toggles controls between read only and editable.

Syntax
VB 
'Declaration
Public Sub MakeReadOnly( _
        frm As imsForm, _
        bReadOnly As Boolean _
)

Parameters

frm
Type: imsForm
The form that owns the controls to toggle.
bReadOnly
Type: System.Boolean
If TRUE the controls will be read only, otherwise their previous state is restored.
Remarks

Only works if the first call has the ReadOnly flag set to TRUE. Then it will make most of the controls read only right after storing their prior state. Then when called again with the ReadOnly flag set to FALSE it will restore the controls prior read only state.

Example

This method works best with a helper method in the host form. This helper method (called MakeFormReadOnly) will ensure that MakeReadOnly will not be called first with a bReadOnly parameter of FALSE. It will also ensure that MakeReadOnly does not get called twice with the same value since this will overwrite the stored settings of the controls if called twice in a row with the bReadOnly parmeter set to TRUE.

You simply call this helper method passing in the desired state of read only and it will handle the rest.

VB 
Private mbMarkedReadOnly As Boolean? = Nothing


Private Sub MakeFormReadOnly(ByVal bReadOnly As Boolean )

    Dim bToggleOK As Boolean


    Select Case bReadOnly
        Case True
            If mbMarkedReadOnly Is Nothing OrElse Not mbMarkedReadOnly Then
                bToggleOK = True
            End If
        Case False
            If mbMarkedReadOnly IsNot Nothing AndAlso mbMarkedReadOnly Then
                bToggleOK = True
            End If
    End Select

    If bToggleOK Then
        UI.MakeReadOnly(Me, bReadOnly)
        mbMarkedReadOnly = bReadOnly
    End If

End Sub
