FillUseClassCombo Function

Populates the UseClass combobox.

Syntax
VB 
'Declaration
Private Sub FillUseClassCombo
Remarks

Fills the combobox based on eItemUseClass.

Example

Creates a DataTable, from an enumeration then sets the datasource of the combobox.

VB 
Private Sub FillUseClassCombo()

    Dim eUseClass As Enumerations.eUseClass
    Dim DR As DataRow
    Dim DT As DataTable


    DT = New DataTable(Constants.UseClassColumn)
    DT.Columns.Add(New DataColumn(mconUseClassColumnHeader, Type.GetType("System.String")))

    For Each eUseClass In [Enum].GetValues(GetType(Enumerations.eUseClass))
        DR = DT.Rows.Add
        DR(0) = eUseClass.ToString
    Next

    cboUseClass.ListView.DataSource = DT

End Sub
