SendAdministratorEMail Method

Sends an e-mail to the admisistrator.

Syntax
VB 
'Declaration
Public Shared Sub SendAdministratorEMail( _
        eEmailType As eEmailForAdministration _
) As Boolean

Parameters

eEmailType
Type: RotairTypes.Enumerations.eEmailForAdministration
The specific type of administrator mail.

Return Value

Type: System.Boolean
TRUE if no problems were found.
Remarks

Sets all the placeholders needed for administrator mail then calls the SendClassBasedEmail function.

Note: in order to find out that an e-mail address is missing the function SendClassBasedEmail would have already been called once. If the administrator's e-mail address is not in the database we would end up with a loop untill we got a stack overflow. The flag mbSendAdministratorEMail ensures that this does not happen.

Example

The example assumes we have already popualted the various variables. If we did not get an e-mail address then an e-mail is sent to the administrator, otherwise the e-mail destined for the employee goes out.

VB 
'Check to see if we have the employee's emial address.
If strEmailAddress = String.Empty Then
    'No email address.
    SendAdministratorEMail(Enumerations.eEmailForAdministration.EmployeeMissingEmail)
Else
    'Send email.
    SendClassBasedEmail(strEmailType, Placeholders)
End If
