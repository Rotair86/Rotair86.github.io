Installation Requirements

These steps are required when a new installation is performed.

Add a Current Install.xml

A Current Install.xml is required for the system to recognize when it is in production or in the sandbox. This is necessary for:
  • Sending automated e-mails
  • Updating / displaying vendor RFQ responses
Automated mails do not get sent to the recipients when in the sandbox. They get sent to the currently logged on user.
When viewing RFQs on the sandbox the user has the choice to view test or production.
