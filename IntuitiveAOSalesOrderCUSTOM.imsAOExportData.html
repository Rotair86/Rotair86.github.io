Customizations

Custom Fields

Added controls for:

  • RI_NoQuote and RI_DD250Reqd.
  • From tblExDpScreen added DeniedParty, ScreenBy and ScreenDate.
  • From tblRI_Note displays Note_Body from the latest record, if there is one.

Informational Footer

The footer has been transitioned to a display of certain key information that is normally visible on a specific tab so that it is always visible.

A label gets displayed for the following:

  • RI_NoQuote is TRUE; displays "No Quoting".
  • DeniedParty is TRUE; displays "DP Stop".
  • Records exist in tblRI_Note; displays "Notes Exist".

Notes

The last note entered, if there is one, is displayed in the Collection Worksheet Notes textbox, which was for field CUS_CollectionNotes, has been renamed to Last Note.

Notes for a customer can be added by clicking on the Notes link label in the task shade. They also can be entered by double-clicking on the Last Note textbox.


Screening

A customer can only be edited if they have a successful screening. This is determined by calling ScreenCustomer. If the customer fails to pass the screening the user is presented with the option to screen the customer via the Screening dialog.


Debugging

When the notes form is displayed the xml that gets passed gets written out to a file.


Notes

When the user opens the notes form the XML that is passed to the form is written to a file.
