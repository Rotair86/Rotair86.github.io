GatAccess Method

Puts Access on a control.

Syntax
VB 
'Declarations
Public Function GatAccess( _
        hwnd As IntPtr _
) As System.Diagnostics.Process

Parameters

hwnd
Type: IntPtr
Pointer or handle to a container.

Return Value

Type: Process
Class representing the process owned by Access if found.
Remarks

Will go through all the running processes and will set the parent of any process matching MSACCESS to point to the passed in handle. The handle can be any container control like a form, panel, picturebox, ect.

Example

The example will put the Access application inside a form.

VB 

Rotair_GeneralFunctions.Windows.GatAccess(Me.Handle)
