Parameters

strMessageID
Type: System.String
Key used to find the message in the table.

Type: MessageBoxIcon
Determines the icon to display.
strTitle
Type: System.String
Text for the title bar.
strMessageSuffix
Type: System.String
Text to append to the end of the message.
strMessageStrings()
Type: System.String
An array of parameters to subtitute for tokens in the message.
btnsUserButtons
Type: MessageBoxButtons
Determines what buttons to display.

Return Value

Type: DialogResult
Indicates what button was clicked.
Remarks

Displays a message from MSG_English based on the passed in key.

Example

The example checks whether an item has had its use class determined. If not then it displays a message with the stop sign icon. This particular message displays the item id in the body of the message so this information is passed in as the last argument.

VB 
If Not imsAO_UseClass.IsUseClassDetermined("57356-001")Then
    UI.DisplayMessage("MSG_Rotair_UndeterminedUseClass", _
                    MessageBoxIcon.Stop, _
                    "Undetermined UseClass", _
                    String.Empty, _
                    New String() {"57356-001"})
End If
