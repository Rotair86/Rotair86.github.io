InitializeListViewUseClass Function

Creates a column for the UseClass combobox.

Syntax
VB 
'Declaration
Private Sub InitializeListViewUseClass
Remarks

The imsCombobox uses a listview to display multiple columns. This function will add a column to the listview.

Example

Creates a column and sets that column to have the selected value displayed in the textbox portion.

VB 
Private Sub InitializeListViewUseClass()

    Dim imsListViewColumn As imsColumnHeader
    Dim imsDisplayField As IntuitiveForms.Binding.imsFormattedFieldBinder


    'imsCombobox
    With cboUseClass

        'Column 1
        imsListViewColumn = New imsColumnHeader
        With imsListViewColumn
            .ValueBinder.FieldName = Constants.UseClassColumn
            .Text = "Use Class"
            .Width = 160
        End With
        .ListViewColumns.Add(imsListViewColumn)

        'Display field from selected row
        imsDisplayField = New IntuitiveForms.Binding.imsFormattedFieldBinder
        With imsDisplayField
            .FieldName = "Use Class"
        End With
        .Display.Fields.Add(imsDisplayField)

    End With

    'Don't know what this does, but it is required.
    .KeyField = Constants.UseClassColumn

End Sub
