MainAO.Saving Event

Sets the Item id and record id for the subdomains.

Syntax
VB 
'Declaration
Protected Overrides Sub MainAO_Saving
Remarks

Only if we are saving a new item do we set the ids.

Example

Sets the item id and record id of the ItemUseClass subdomain if we are saving a newly created item then calls the base.

VB 
Protected Overrides Sub MainAO_Saving( _ 
    ByVal semder As System.Object, _
    ByVal e As System.ComponentModel.CancelEventArgs _
    )

    Dim UseClass As imsAO_UseClass
    Dim colUseClass As imsAO_UseClass_Collection


    'We only need to update these fields on a new item.
    If MainAO.AO.DocumentState = IntuitiveAppObjectFramework2.aoObjectState.NEW_DOCUMENT Then
        'Subdomains are stored in collections even if there is a 1 to 1 relationship.
        colUseClass = CType(MainAO.AO.Subdomains("ItemUseClass"), imsAO_UseClass_Collection)
        'We can jump right to the first element since there can only be one item in the collection.
        UseClass = CType(colUseClass(0), imsAO_UseClass)
        'Set the ids.
        UseClass.SetValue("ItemID", MainAO.AO.GetString("IMA_ItemID"))
        UseClass.SetValue("ItemRecordID", MainAO.AO.GetString("IMA_RecordID"))
    End If

    MyBase.MainAO_Saving(sender, e)

End Sub
