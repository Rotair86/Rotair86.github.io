Count Property

Returns the number of objects the collection contains.

Syntax
VB 
'Declaration
Public Property Count As Integer

Remarks

This property returns the number of placeholder objects contained by the placeholders collection.

Example

The example displays a message if there aren't any items in the collection.

VB 
Private Function CheckPlaceholdersCollection(colPH As rotPlaceholders) As Boolean

    Dim rc As Boolean


    If colPH.Count < 1
        rc = False
        Messagebox.Show("Nothing in the collection.")
    Else
        rc = True
    End If

    Return rc

End Function
