eEmailForSalesOrder Enumeration

Types of auto-generated e-mail based on sales orders.

Members
  Member name Description
None Default if no value is assigned.
ExistingExportOrderDefenseArticle Defense article(s) have been added to a sales order that may require an export license.
ExistingExportOrderDualUse Dual use item(s) have been added to a sales order that may require an export license.
ExistingExportOrderWithOutSubmittedLicense A sales order that may require an export license that has not been submitted had some field(s) changed that may require changes to the export license application.
ExistingExportOrderWithSubmittedLicense A sales order that requires an export license and the application has been submitted had some field(s) attempted to be changed that affects the export license.
NewExportOrderDefenseArticle A sales order was created that has defense article(s) and may require an export license.
NewExportOrderDualUse A sales order was created that has dual use item(s) and may require an export license.
