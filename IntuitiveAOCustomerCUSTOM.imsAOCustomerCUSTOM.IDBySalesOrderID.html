IDBySalesOrderID Method

Returns a customer id from a sales order id.

Syntax
VB 
'Declaration
Public Function IDBySalesOrderID( _
        strSalesOrderID As String _
) As String

Parameters

strSalesOrderID
Type: System.String
The sales order id.

Return Value

Type: System.String
The customer id.
Remarks

Returns a customer id based on a sales order id.

Example

The example retrieves the customer id using the sales order id on the shipment card.

VB 
Dim strCustomerID As String
Dim strSalesOrderID As String

strSalesOrderID = KeyInputSO_fndSalesOrderFind.SelectedID
strCustomerID = msAO_Customer_CUSTOM.IDBySalesOrderID(strSalesOrderID)
