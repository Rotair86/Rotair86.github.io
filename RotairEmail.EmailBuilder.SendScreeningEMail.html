SendScreeningEMail Method

Sends an e-mail when a customer or vendor that are marked "denied" get added to a domain. Also will send an e-mail if a domain associated with a "denied party" is opened.

Syntax
VB 
'Declaration
Public Shared Function SendScreeningEMail(Of PartyKey)( _
        ePartyType As ePartyType, _
        PartyID As PartyKey, _
        eSourceAction As eScreenAction, _
        strSourceForm As String _
) As Boolean

Type Parameters

PartyKey
The type of key used. Can be a string or guid.

Parameters

ePartyType
Type: ePartyType
The specific type which can be customer or vendor.
PartyID
Type: PartyKey
Key for the customer or vendor.
eSourceAction
Type: eScreenAction
Indicates what caused the screening. Can be adding a party to a domain (like sales order or purchase order) or opening an item liked to a party.
strSourceForm
Type: System.String
The form where the screening took place.

Return Value

Type: System.Boolean
TRUE if no problems were found.
Remarks

Sets all the placeholders needed and determines the e-mail type then calls the SendClassBasedEmail function.

Example

This function will get passed a SalesOrder object and customer id. It will then check to see if the sales order has a denied customer. If it does then it will send the e-mail.

VB 
Private Sub CheckForADeniedCustomerAndSendEmailIfNeeded(ByVal strCustomerID As String, ByVal SalesOrder As imsAO_SalesOrder_CUSTOM)


    'Send an e-mail when opening a sales order for a customer marked as a denied party.
    If SalesOrder.ScreenResults = Enumerations.eScreeningRC.Denied_Party Then
        'Send an e-mail indicating a sales order was opened pertaining to a denied customer.
        If SendScreeningEMail(Enumerations.ePartyType.Customer, strCustomerID, SalesOrder.ScreenAction, "'Sales Order Card'") Then
            'Success.
            MessageBox.Show("E-mail sent.")
        Else
            'Possible failure.
            MessageBox.Show("E-mail indicating a sales order was opened for a denied customer may not have went out.")
        End If
    End If

End Sub
