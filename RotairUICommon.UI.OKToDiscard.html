OKToDiscard Method

Displays a message if the object is dirty.

Syntax
VB 
'Declaration
Public Function OKToDiscard( _
        RotairObject As rotAppObject _
) As Boolean

Parameters

RotairObject
Type: rotAppObject
The object to check.

Return Value

Type: System.Boolean
TRUE if the object is not dirty OR the user wishes to discard the changes.
Remarks

Displays a message asking the user if they wish to discard the changes if the object is dirty. Returns TRUE if the object is dirty and the user clicked "Yes" OR the object is not dirty.

Example

This example is from the cancel button of the CageCode form. It disposes the current object and clears the form only if the functions returns TRUE.

VB 
Private Sub btnCancel_Click(ByVal sender As System.Object, ByVal e As System.EventArgs)Handles btnCancel.Click

    If UI.OKToDiscard(mCageCode) Then
        mCageCode = Nothing
        ClearForm()
    End If

End Sub
