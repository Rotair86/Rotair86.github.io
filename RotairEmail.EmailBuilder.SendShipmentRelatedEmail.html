SendShipmentRelatedEmail Method

Sends various e-mails including mails relating to screening and the export license.

Syntax
VB 
'Declaration
Public Shared Function SendShipmentRelatedEmail( _
        eEmailType As eScreeningRC, _
        strSalesOrderID As String, _
        strEndUser As String, _
        strCustomerID As String _
) As Boolean

Parameters

eEmailType
Type: RotairTypes.Enumerations.eScreeningRC
The screening results determine what kind of mail gets sent.
strSalesOrderID
Type: System.String
The sales order id associated with this shipment.
strEndUser
Type: System.String
The end user this shipment is destined for.
strCustomerID
Type: System.String
The customer who is initially getting this shipment.

Return Value

Type: System.Boolean
TRUE if no problems were found.
Remarks

Sets all the placeholders needed and determines the e-mail type then calls the SendClassBasedEmail function.

Example

This example checks the screening result and if it is not OK then passes it to the function which will use it to determine what kind of mial to send.

VB 

'Check to see if we need to send a screening e-mail.
If meScreenStatus <> Screened_OK Then
    'Some type of screening issue the functions will interpet and send the appropriate mail.
    SendShipmentRelatedEmail(meScreenStatus, strSalesOrderID, strEndUser, strCustomerID)
End If
