DisplayEMailFailure Method

Displays a messagebox indicating that an e-mail may have failed to go out.

Syntax
VB 
'Declarations
Public Sub DisplayEMailFailure( _
)

Public Sub DisplayEMailFailure( _
        strMessageSuffix As String _
)

Parameters

strMessageSuffix
Type: System.String
Text to append to the message.
Remarks

The message is displayed when an e-mail message function does not return success. It is conceivable that the SMTP service can still send a message but not return with success instead return with a warning. The e-mail sending is working so it was not deemed necessary to anticipate any of the errors that can occur. This is a general message displayed if the call does not return 100% success.

Example

The e-mail message for failure is displayed is displayed if the screening e-mail function does not return success. The error form the EmailBuilder class is appended to the message (if there is one).

VB 
If Not rotEmailBuilder.SendScreeningEMail(ePartyType.Customer, "623454", SalesOrder.ScreenAction, "'Sales Order Card'") Then
    RotairUI_Common.CustomMessage.DisplayEMailFailure(rotEmailBuilder.LastError)
End If
