CheckAllRequiredFields Function

Gets the parents key and sets the foriegn key field in the child.

Syntax
VB 
'Declaration
Public Overrides Function CheckAllRequiredFields( _
    ) As imsReturnData
Remarks

Establishes the link between the parent and the child, contrary to its name.

Example

This function calls a function that sets a modular variable to the parent's key. It then takes this value and sets a field in the child.

VB 
Public Overrides Function CheckAllRequiredFields( _
    ) As imsReturnData

    'Get the parents key.
    GetParentKeyFields()
    r2d2.Merge(SetValue(True, mChildsParentKeyFieldName, mParentKey))
    If r2d2.AnyAppErrs Then Return r2d2

    Return MyBase.CheckAllRequiredFields()

End Function
