LastError Property

Provides access to error text.

Syntax
VB 
'Declarations
Public Shared Property LastError As String

Remarks

Holds the last error that was trapped. An error can be stored here that happened in a much earlier function. So it is recommended to check this property to ensure it is clear before calling any functions where an error may be expected.

Example

The example will call a function and if it does not return successfully will display the contents of LastError in a message box.

VB 
eCustomerTypeScreen As Enumerations.eCustomerTypeScreenResult

eCustomerTypeScreen = Rotair_ExportManagement.ExportManagement.CustomerTypeScreen("002436")
If eCustomerTypeScreen = Enumerations.eCustomerTypeScreenResult.ScreeningErrorCondition Then
    MessageBox.Show("Function did not return success." & Environment.NewLine & _
            Rotair_ExportManagement.ExportManagement.LastError)
End If
