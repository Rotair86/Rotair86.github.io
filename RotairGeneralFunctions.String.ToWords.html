ToWords Method

Returns a string of wrods separated by spaces.

Syntax
VB 
'Declarations
Public Function ToWords( _
        strIdentifier As String
) As String

Parameters

strIdentifier
Type: System.String
A string of words separated by uppercase characters.

Return Value

Type: System.String
A string of words separated by spaces.
Remarks

Separates a string into separate words where the start of the words are indicated by uppercase.

Example

The example displays an enumeration as words.

VB 
'Show an enumerration as words.
MessageBox.Show(Rotair_GeneralFunctions.ToWords(Rotair_Types.Enumerations.eCustomerTypeScreenResult.NoCustomerType.ToString())
