Tokens Enumeration

Represents tokens that are used for values that are unknown until the report is printed.

Members
  Member name Description
None Indicates that the variable that is defined as this type has not been assigned.
Machine The name of the machine the code is running on. Example: R0177
Now Current date and time. Example: 5/17/2012 2:58:53 PM
Page Current page. Example: 1
Pages Total number of pages (including all tables). Example: 12
TimeLong Current time. Example: 2:58:53 PM
TimeShort Current time without the seconds. Example: 2:58 PM
TodayCustom Intended for a custom format string. Not implemented. Example: {Undefined}
TodayLong Current date with the day and month spelled out. Example: Thursday, May 17, 2012
TodayShort Current date. Example: 5/17/2012
User Current logged on user. Example: JohnM
Example

This shows how to print the page number alnong with the total number of pages.

VB 
'Using the defaults, the string looks like this: Printing.Text = "Page {[Page]} of {[Pages]}"
'To reduce maintenance costs (among other benefits) it is strongly recommended to avoid hardcoding (use the constants as in the example)"
Printing.Text = "Page " & Printing.TokenOpen & rotHC_Reporter.Tokens.Page.ToString & Printing.TokenClose & _
     " of " & Printing.TokenOpen & rotHC_Reporter.Tokens.Pages.ToString & Printing.TokenClose

Assuming we are on page 1 of a 12 page report, prints:

Page 1 of 12
