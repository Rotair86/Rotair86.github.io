ShowData Method

Copies a datset to an Excel document.

Syntax
VB 
'Declaration
Public Sub ShowData( _
        eWhich As eExcelReport, _
        DT As DataTable _
)

Parameters

eWhich
Type: eExcelReport
Type of report.
DT
Type: DataTable
Data to copy to the spreadsheet.
Remarks

Copies the datatable to the spreadsheet. The first argument determines colmn formatting.

Example

This example copies a bookings report from a grid to an Excel spreadsheet.

VB 
XL = New rotExcel("C:\My Documents\Spreadsheet.xls")
XL.ShowData(Enumerations.eExcelReport.Booking, Grid.DataSource)
