NewSetup Method

Called upon object instantiation to set defaults.

See NewSetup in the base.
Syntax
VB 
'Declaration
Public Overrides Function NewSetup( _
    ) As imsReturnData

Return Value

Type: imsReturnData
Holds messages.
Remarks

Only override this method if you need to instantiate custom subdomains or set defaults for actual or logical fields on object creation.

Example

Here the NewSetup method is overridden to create a new guid for the record id field and prepare two subdomains, 'ItemSupplemental' and 'ItemUseClass'.

VB 
Protected Overrides Function NewSetup() As imsReturnData

    Dim ao As imsAppObject
    Dim r2d2 As New imsReturnData()


    'Call the base to set the standard defaults.
    r2d2.Merge(MyBase.NewSetup)
    'Sets the field IMA_RecordID to a new guid.  This is already done in the base.
    '    It is only replicated here for example purposes.
    r2d2.Merge(SetValue(True, "IMA_RecordID", Guid.NewGuid))
    'Must be done for custom subdomains.
    ao = Me.ItemSupplemental_Collection.NewObject()
    ao = Me.ItemUseClass_Collection.NewObject()

    Return r2d2

End Function
