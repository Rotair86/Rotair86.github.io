SendSalesOrderRelated Method

Sends various e-mails including mails relating to defense articles, dual use items and export licenses.

Syntax
VB 
'Declaration
Public Shared Function SendSalesOrderRelated( _
        eEmailType As eEmailForSalesOrder, _
        strSalesOrderRecordID As String, _
        strCustomerRecordID As String _
) As Boolean

Parameters

eEmailType
Type: RotairTypes.Enumerations.eEmailForSalesOrder
The specific type of sales order mail.
strSalesOrderRecordID
Type: System.String
The sales order record id (guid).
strCustomerRecordID
Type: System.String
The customer record id (guid) relating this sales order.

Return Value

Type: System.Boolean
TRUE if no problems were found.
Remarks

Sets all the placeholders needed and determines the e-mail type then calls the SendClassBasedEmail function.

The record ids (guids) are passed as strings but should eventually be changed over to guids.

Example

This function will get passed a SalesOrder object and pass it to a couple of functions that will return TRUE if the sales order contains any defense articles or dual use items. If any items are found an e-mail is sent so export licenses can be prepared.

VB 

Private Sub CheckForItemsRequiringExportLicense(ByVal SalesOrder As imsAO_SalesOrder_CUSTOM)

    'Does the sales order contains a defense article?
    If HasDefenseArticle(SalesOrder) Then
        'Send an e-mail.
        SendSalesOrderRelated(Enumerations.eEmailForSalesOrder.NewExportOrderDefenseArticle, _
            SalesOrder.GetString("SOM_RecordID"), _
            SalesOrder.GetString("SOM_CUS_RecordID"))
    End If

    'Does the sales order contains a dual use item?
    If HasDualUseItem(SalesOrder) Then
        'Send an e-mail.
        SendSalesOrderRelated(Enumerations.eEmailForSalesOrder.NewExportOrderDualUse, _
            SalesOrder.GetString("SOM_RecordID"), _
            SalesOrder.GetString("SOM_CUS_RecordID"))
    End If

End Sub
