frmTestForm form

Form used to test the report writer rotHC_Reporter by creating dummy data.


Sample of TestForm and data.

To use this form follow the following steps.

  1. Enter the number of columns you want in the first textbox.
  2. Click the 'Set Cols' buttom.
  3. Optional: You can set the minumom and/or maxumom size of the data (applicable only to alpha, numeric or alpha-numeric). The percent of data that has a null value and the datatype. Choices of datatype are:
    • Unspecified - (has not been coded for; does not work.)
    • Alphabetic - letters 'a' through 'z'.
    • Alphanumeric - letters 'a' through 'z' and digits 0 through 9.
    • Date - in the form m/d/yyyy after 1/1/1950.
    • DateTime - a date in the fashion you get from using Date with a Time appended to the end of it.
    • Numeric - digits 0 through 9.
    • Time - a time in the format hh:mm.
  4. Enter the number of rows you want in the second textbox.
  5. Click the 'Set Rows' buttom.
  6. Click the 'Preview' buttom to get a print preview of the report. You can also print from the preview.
  7. Click the 'Print' buttom to print the report.
  8. The 'Add Table' buttom is intended to add a table to the dataset. The report writer supports this. The TestForm does not (button has no code behind it).
Example

This is the example code used to demonstrate and test teh report writer.

VB 
With Printing

    'Customize HERE


    'The first header (or subheader or footer) does not need to be indexed.
    With .Header.CenterText
        .Color = Drawing.Color.Aqua
        .PrintFrequency = rotHC_Reporter.HeaderFooterOptions.PrintOnce
        .Size = 22
        .Text = "Denied Party (Customer) Order History and Status"
    End With
    'Any additional headers past the first (including subheaders and footers) need to be indexed.
    .Header(2).CenterText.PrintFrequency = rotHC_Reporter.HeaderFooterOptions.PrintEveryPage
    .Header(2).CenterText.Text = "Denied Party 011300          Airrod whatever"
    .Header(2).LeftText.PrintFrequency = rotHC_Reporter.HeaderFooterOptions.PrintEveryPage
    .Header(2).LeftText.Text = "Left ----"
    .Header(2).RightText.PrintFrequency = rotHC_Reporter.HeaderFooterOptions.PrintEveryPage
    .Header(2).RightText.Text = "---- thgiR"

    'A subheader is associated with a table only noticed in a multi-table report.  Otherwise appears just lie a header.
    With .SubHeader.CenterText
        .Color = Drawing.Color.Brown
        .PrintFrequency = rotHC_Reporter.HeaderFooterOptions.PrintEveryPage
        .Size = 16
        .Text = "Orders"
    End With

    'The first column header does require an index.  The first line of a multi-line column header does not.
    .ColumnHeaders(1).TextLine = "Numero"
    .ColumnHeaders(1).TextLine(2) = "Uno"
    'Columns can be skipped if there is no need to customize them.
    .ColumnHeaders(3).TextLine = "Header 3"
    .ColumnHeaders(3).TextLine(2) = "--------"
    .ColumnHeaders(3).Color = Drawing.Color.Red
    .ColumnHeaders(3).Size = 24
    .ColumnHeaders(3).Style = Drawing.FontStyle.Strikeout
    .ColumnHeaders(5).Align = rotHC_Reporter.Alignment.Left
    .ColumnHeaders(6).TextLine = "Hex"
    .ColumnHeaders(7).TextLine(2) = "Sept"
    .ColumnHeaderGroups(0).AutoFitColumns = True
    .ColumnHeaderGroups(0).HideColumnHeaders = False
    .DetailColumn(2).Color = Drawing.Color.Green
    .DetailColumn(2).Size = 24
    .DetailColumn(5).Align = rotHC_Reporter.Alignment.Right

    'Tokens can be used in any header or footer.
    With .Footer.LeftText
        .Color = Drawing.Color.Blue
        .Font = "Times New Roman"
        .PrintFrequency = rotHC_Reporter.HeaderFooterOptions.PrintEveryPage
        .Size = 9
        .Text = Printing.TokenOpen & rotHC_Reporter.Tokens.TodayLong.ToString & Printing.TokenClose
    End With
    With .Footer(1).RightText
        .Color = Drawing.Color.Blue
        .Font = "Times New Roman"
        .PrintFrequency = rotHC_Reporter.HeaderFooterOptions.PrintEveryPage
        .Size = 9
        .Text = "Page " & Printing.TokenOpen & rotHC_Reporter.Tokens.Page.ToString & Printing.TokenClose & _
             " of " & Printing.TokenOpen & rotHC_Reporter.Tokens.Pages.ToString & Printing.TokenClose
    End With

    'Pass in the data.
    .Data = mDS

    'Statistics are for debugging.
    .ShowStatistics = True

End With
