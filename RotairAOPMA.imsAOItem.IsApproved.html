IsApproved Method

Retiurns TRUE if a part is PMA approved.

Syntax
VB 
'Declaration
Public Function IsApproved( _
        strItemID As String _
) As Boolean

Parameters

strItemID
Type: System.String
The item id to check for approval.

Return Value

Type: System.Boolean
TRUE if the item is PMA approved.
Remarks

Checks the status of an item to see if it is approved.

Example

The example will display a custom message if the item is not PMA approved.

VB 

If Not imsAO_Item.IsApproved("0-089-177-01") Then
    RotairUI_Common.UI.DisplayMessage("MSG_Rotair_SalesOrderPMANotApproved", _
                                        MessageBoxIcon.Stop, _
                                        "PMA Not Approved", _
                                        String.Empty, _
                                        New String() {"0-089-177-01"})
End If
