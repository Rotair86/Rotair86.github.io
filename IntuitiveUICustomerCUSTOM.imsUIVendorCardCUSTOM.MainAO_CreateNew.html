MainAO_CreateNew Function

Disables the notes linklabel.

Syntax
VB 
'Declaration
Protected Overrides Function MainAO_CreateNew ( _
        vendorID As String _
) As Boolean

Parameters

vendorID
Type: System.Object
Id for the new vendor.

Return Value

Type: System.Boolean
Returns TRUE if the vendor is succeddfully instantiated.
Remarks

Disables the notes linklabel on the task shade used to bring up the notes form which requires to be in the context of a vendor before allocating memory for a new vendor.

Example

Simply disables a control before passing control to the base. This function doesn't get called directly.

VB 
Protected Overrides Function MainAO_CreateNew( _ 
    ByVal vendorID As String _
    ) As Boolean

    lnkNotes.Enabled = False
    Return MyBase.MainAO_CreateNew(vendorID)

End Function
