Remarks

This assembly is part of the build process and is the first one to be built. Once it is built it is then executed. It runs, reads some tables from the database and builds enumerations based on the data. This allows someone, besides the programmer to add options. The user will add choices via the ERP application. Then when the programmer runs the build script the enumerations will be created. All that is left to do is reference the RotairDynamicEnums dll if not already done so and the new enumerations will be visible via intellisense.
