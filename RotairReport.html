Remarks

This report manager can produce table driven reports or produce a report based on a dataset.

It works with for table driven reports and to produce the output.

Examples

Below is an example of a creating a report for purchases made by a customer in a preview window. This report has already been defined in the database.

VB 
'A report named "CustomerPurchses" is assumed to be created in the database.
Dim Report As Rotair_Report.rotReport
'Parameters for the SQL are passed in after the report name.  They are substituted in the order that they are passed.
'     There isn't a limit to the number of parameters as long as they are handled by RotairDataAccess.DataSetCustom.Generic.
Report = New Rotair_Report.rotReport("CustomerPurchses", strCustomerID)
With Report
    .PopulateData()'Calls RotairDataAccess.DataSetCustom.Generic to populate the dataset.
    .CopyDataToReporter()'Copies data from rotAO_Report to rotHC_Report.
    .PrintPreview()'Opens a window with a preview and the option to print.
End With
