Remarks

This is where most of the work is done.

  • Create a new dynamic assembly
  • Create a new module
  • Call function(s) to add an enumeration to the module
  • Save the new assembly

Adding enumerations to the assembly is as simple as adding a new function in Builder similar to the others. Add a function in

DataAccess to return a dataset and youre done. Add a reference to any project you wish to use the new enumeration.
