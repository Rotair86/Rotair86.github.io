Save Method

Called when saving an object.

See Save in the base.
Syntax
VB 
'Declaration
Public Overrides Function Save( _
    ProcessNow As Boolean
    ) As imsReturnData

Parameters

ProcessNow
Type: System.Boolean
Undetermined.
Remarks

Only override this method if you need to do anything immediately before or after the save.

Example

Subdomains need to be active or they won't save. The subdomains 'ItemSupplemental' and 'ItemUseClass' are activated in the example so they will be saved. There may be a better way or better place to do this but I haven't found it yet.

VB 
Protected Overrides Function Save() As imsReturnData

    Dim r2d2 As New imsReturnData()


    'Activate the subdomains before calling Save.
    Me.SubdomainActivate("ItemSupplemental")
    Me.SubdomainActivate("ItemUseClass")
    'Now do the actual saving.
    r2d2 = myBase.Save(ProcessNow)

    Return r2d2

End Function
