ShowNotes Function

Displays the notes form in the context of the current purchase order.

Syntax
VB 
'Declaration
Public Sub ShowNotes( _
    xmlDoc As XmlDocument
)

Parameters

strID
Type: System.Xml.XmlDocument
Ids used to look up the note(s).
Remarks

Calls imsForm.LaunchMenuItem to display the Notes form based on the passed in xml which can be based on the header or a line.

Example

Displays the notes form based either on the purchase order or a line item. If bHeader is TRUE then the note(s) are based on the purchase order. If bLineItem is TRUE then the note(s) are based on a line item. PurchaseOrderAO represents the AppObject for a purchase order and PurchaseOrderLineAO represents an AppObject for a line item which is a subdomain of the purchase order AO. It isn't always straight forward getting access to the AOs. Shortcuts are presented here to illustrate the function since it is beyond the scope of this help to document Intuitive's operation.

VB 
Dim xmlDoc As XmlDocument

If bHeader Then
    'Get the purchase order id and record id.
    xmlDoc = rotAO_NoteType.BuildPOXML( _
        PurchaseOrderAO.GetString(imsAO_PurchaseOrder.fPOM_PurchOrderID), _
        PurchaseOrderAO.GetString(imsAO_PurchaseOrder.fPOM_RecordID))
ElseIf bLineItem Then
    'Get the purchase order id, record id, line number and line record id.
    xmlDoc = rotAO_NoteType.BuildParentAndChildXML( _
        PurchaseOrderAO.GetString(imsAO_PurchaseOrder.fPOM_PurchOrderID), _
        PurchaseOrderAO.GetString(imsAO_PurchaseOrder.fPOM_RecordID), _
        PurchaseOrderLineAO.GetString(imsAO_PurchaseOrderLine.fPOI_POLineNbr), _
        PurchaseOrderLineAO.GetString(imsAO_PurchaseOrderLine.fPOI_RecordID), _
        'PurchaseOrderLine")
End If

If xmlDoc IsNot Nothing Then
    ShowNotes(xmlDoc)
End If
