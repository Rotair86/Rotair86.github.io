CanNotSaveMessage Method

Returns an enumeration representing a message.

Syntax
VB 
'Declarations
Public Function CanNotSaveMessage( _
        eWhich As eScreeningRC _
) As MSG_ScreeningFailure

Parameters

eWhich
Type: eScreeningRC
Return code from a screening function.

Return Value

Type: MSG_ScreeningFailure
A message enumeration.
Remarks

Returns an enumeration modeled after the intuitive MSG enumeration which use an attribute to store a string with an enumeration.

Example

The example will add a message to the imsReturnData object to be displayed in the UI.

VB 
Dim r2d2 As New imsReturnData()

If ScreenResults <> Enumerations.eScreeningRC.Screened_OK Then
    r2d2.ApplicationErrors.Add(Scalar.CanNotSaveMessage(ScreenResults))
End If
