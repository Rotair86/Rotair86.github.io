NameByID Method

Returns the vendor's name.

Syntax
VB 
'Declaration
Public Function NameByID(Of VendorKey)( _
        VendorID As VendorKey _
) As String

Type Parameters

VendorKey
The type of vendor key used. Can be a string or guid.

Parameters

VendorID
Type: VendorKey
Key for the vendor.

Return Value

Type: System.String
The vendor's name.
Remarks

Returns the vendor's name by either the vendor id or record id.

Example

The example will retrieve the vendor name two different ways.

VB 
Dim strName As String


strName = imsAO_Vendor_CUSTOM.NameByID(MainAO.AO.GetGuid("VEN_RecordID"))
strName = imsAO_Vendor_CUSTOM.NameByID(MainAO.AO.GetString("VEN_VendorID"))
