DefineSubdomains Method

Required for an AppObject to load child data defined in the AppObject_MetadataHeader.

See DefineSubdomains in the base.
Syntax
VB 
'Declaration
Protected Overrides Sub DefineSubdomains()
Remarks

Required to load a custom subdomain automatically when the parent loads. Instructions on creating a subdomain can be found here.

Example

Here is the DefineSubdomains method overridden to add two children, one is 'ItemSupplemental' and the other 'ItemUseClass.

VB 
Protected Overrides Sub DefineSubdomains()

    'Add the supplemental subdomain.
    MyBase.SubdomainDefs.Add("ItemSupplemental", _
            requiredForDelete:=True, _
            requiredForAdd:=True, _
            relationshipToParent:=imsRelationshipToParent.OneToOne)
    'Add the use class subdomain.
    MyBase.SubdomainDefs.Add("ItemUseClass", _
            requiredForDelete:=True, _
            requiredForAdd:=True, _
            relationshipToParent:=imsRelationshipToParent.OneToOne)
    'Call the base so it can add any subdomains that it needs to.
    MyBase.DefineSubdomains()

End Sub
