CustomerTypeScreen Method

Provides customer type information.

Syntax
VB 
'Declarations
Public Function CustomerTypeScreen(Of CustomerKey)( _
        CustomerID As CustomerKey _
) As eCustomerTypeScreenResult

Type Parameters

CustomerKey
The type of customer key used. Can be a string or guid.

Parameters

CustomerID
Type: CustomerKey
Key for the customer.

Return Value

Type: eCustomerTypeScreenResult
The status of the export license.
Remarks

Returns the type of a customer.

Example

The example will display a message with the contents depending on the type assigned to the customer.

VB 
eCustomerType As Enumerations.eCustomerTypeScreenResult

eCustomerType = ExportManagement.CustomerTypeScreen("623454")
Select Case eCustomerType
    Case Enumerations.eCustomerTypeScreenResult.NoCustomerType
        Messagebox.Show("Type was never assigned.")
    Case Enumerations.eCustomerTypeScreenResult.Domestic 
        Messagebox.Show("Customer in the US.")
    Case Enumerations.eCustomerTypeScreenResult.ForeignCustomer
        Messagebox.Show("Customer located in a foriegn country.")
    Case Enumerations.eCustomerTypeScreenResult.Government
        Messagebox.Show("Federal government.")
End Select
