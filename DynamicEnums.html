Remarks

This assembly is used for storing enumerations based on entries in the master controls. To modify simply edit the master control settings and save them. Then run this C:\IntuitiveDev\Bin\EnumBuilder.exe and the assembly will be created. It will then need to be moved to the appropriate directory.
