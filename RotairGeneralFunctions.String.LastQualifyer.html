LastQualifyer Method

Returns the object name from a fully quanified name.

Syntax
VB 
'Declarations
Public Function LastQualifyer( _
        strFullyQualified As String _
) As String

Parameters

strFullyQualified
Type: System.String
A fully qualified name.

Return Value

Type: System.String
The object name.
Remarks

This function returns the object name from a fully quanified name.

Example

The example will return "WindowsApplication1.Form1 - Form1" for a project created with the defaults.

VB 

Messagebox.Show(Me.GetType.ToString() & " - " & LastQualifyer(Me.GetType.ToString())
