frmDialogPhrases Form

Resizeable form used to modify a Phrases field.

Currently there is no way to add new phrases. There will likely be some form to enter them in the future.


Namespace: RotairDialog_Phrases
Assembly: RotairDialog_Phrases.dll
Usage
To use the phrases form the following steps need to be completed:
  1. The form that will request the phrases needs to be added to table tblPhraseForm.
  2. At least one phrase for the requesting form needs to be in the table tblPhrase. One must be added if none exist.
  3. Any phrases in the table tblPhrase that are to be used in the requesting form need a link in table tblPhraseUsage.
  4. Add an enumeration for the form in Rotair_Types.Enumerations.ePhraseForm
  5. Add a reference to RotairDialog_Phrases.dll.
  6. Add a reference to Rotair_Types.dll if there isn't one already.
  7. Declare a variable as a RotairDialog_Phrases.frmDialog_Phrases with events.
  8. Instantiate the variable.
  9. Set the TargetForm which will be used to filter the phrases.
  10. Call the ShowPhrases method.
  11. Handle the event PhrasesSelected.
Steps

1) Adding the form to table tblPhraseForm

SQL 
INSERT INTO tblPhraseForm
	(RecType, PhraseFormID, PhraseFormName, PhraseFormDescription, RevNbr, LastRevBy, LastRevDate)
	VALUES ('A', NewID(), 'frmNewForm', 'New form that uses phrases', 0, 'user', GetDate())


2) Adding a phrase to the table

SQL 
INSERT INTO tblPhrase
	(RecType, PhraseID, PhraseName, Phrase, RevNbr, LastRevBy, LastRevDate)
	VALUES ('A', NewID(), 'XXX', 'Don''t drink the milk, it''s spoiled!', 0, 'user', GetDate())


3) Adding usage

Links the phrase created in step 2 with the form from step 1. The guids supplied were created from the NewID() function.
SQL 
INSERT INTO tblPhraseUsage
	(RecType, PhraseID, PhraseFormID, RevNbr, LastRevBy, LastRevDate)
	VALUES ('A', '84F79E8B-E38E-47A7-9635-FF90F1BD97C8', '3372CD49-F4F7-409C-BC29-5B6364EDB8FC', 0, 'user', GetDate())
