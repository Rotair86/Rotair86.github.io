ShowPhrases Method

Populates the phrase collection and displays the form.

Syntax
VB 
'Declaration
Public Sub ShowPhrases ( _
        strOldPhrase As String _
)

Parameters

strOldPhrase
Type: System.String
The pre-existing phrases.
Remarks

This method will present the phrases form to the user. It will . . .

  1. Gather the phrases from the database based on what gets assigned to the property TargetForm.
  2. Populates the listboxes.
  3. Displays any old phrase(s) passed in.
  4. Shows the form.
  5. Drops down the listbox if no old phrase(s) were passed in.
Example

How to display the phrases form from the purchase order header.

VB 
mfrmPhrases = New frmDialogPhrases
mfrmPhrases.TargetForm = Rotair_Types.Enumerations.ePhraseForm.PurchaseOrder
mfrmPhrases.ShowPhrases(txtPhrases.Text)
