InitializeComponentsEx Function

Adds buttons for the notes and phrases forms in grid cells.

Syntax
VB 
'Declaration
Protected Overrides Sub InitializeComponentsEx
Remarks

Adds a notes button to each grid; line items, miscellaneous lines and work order lines. Also adds a phrases button to the line items grid.

The notes column needs to be added while the phrases use the special instructions field.

Example

Calls the base then adds a notes and phrases button to the line items grid.

Creating a RepositoryItemButtonEdit object automatically creates a button 0 in the buttons collection.

VB 
Protected Overrides Sub InitializeComponentsEx()

    MyBase.InitializeComponentEx()

    '*** Line Items Notes ****
    'Create the button
    mNotesLineItemsButton = New DevExpress.XtraEditors.Repository.RepositoryItemButtonEdit 'Automatically adds button 0
    With mNotesLineItemsButton.Buttons(0)
        .Kind = DevExpress.XtraEditors.Controls.ButtonPredefines.Combo
        .Width = 15
    End With

    'The button needs to be part of the repository items collection.
    ItemLines_grdItemLines.RepositoryItems.Add(mNotesLineItemsButton)
    'We need a reference to the view to trap some events.
    mgvwViewLineItems = CType(ItemLines_grdItemLines.MainView, DevExpress.XtraGrid.Views.Grid.GridView)
    'Adds a column to the grid.
    grdColumn = mgvwViewLineItems.Columns.AddField("Notes")
    With grdColumn
        .AbsoluteIndex = 1
        'Assigns the previously created button to this column.
        .ColumnEdit = mNotesLineItemsButton
        .ShowButtonMode = DevExpress.XtraGrid.Views.Base.ShowButtonModeEnum.ShowOnlyInEditor
        .UnboundType = DevExpress.Data.UnboundColumnType.String
        .Visible = True
    End With

    '*** Line Items Phrases ****
    'Create the button
    mPhrasesLineItemsButton = New DevExpress.XtraEditors.Repository.RepositoryItemButtonEdit 'Automatically adds button 0
    With mPhrasesLineItemsButton.Buttons(0)
        .Kind = DevExpress.XtraEditors.Controls.ButtonPredefines.Combo
        .Width = 15
    End With

    'The button needs to be part of the repository items collection.
    ItemLines_grdItemLines.RepositoryItems.Add(mPhrasesLineItemsButton)
    'Get a reference to an exisitng column.
    grdColumn = mgvwViewLineItems.Columns("PurchaseOrderItemLine.POI_SpecialInst")
    With grdColumn
        'Assigns the previously created button to this column.
        .ColumnEdit = mPhrasesLineItemsButton
        .ShowButtonMode = DevExpress.XtraGrid.Views.Base.ShowButtonModeEnum.ShowOnlyInEditor
    End With

End Sub
