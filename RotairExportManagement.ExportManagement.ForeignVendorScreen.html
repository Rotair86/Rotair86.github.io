ForeignVendorScreen Method

Checks a vendor for foriegn links.

Syntax
VB 
'Declarations
Public Function ForeignVendorScreen(Of VendorKey)( _
        VendorID As VendorKey _
) As Boolean

Type Parameters

VendorKey
The type of vendor key used. Can be a string or guid.

Parameters

VendorID
Type: VendorKey
Key for the vendor.

Return Value

Type: System.Boolean
TRUE if the vendor has any foreign country references.
Remarks

Returns TRUE if the vendor has a foreign address, a foriegn contact or any reference to a foriegn address.

Example

The example will display a message if there is a possibility of a foriegn purchase.

VB 
If ExportManagement.ForeignVendorScreen("176930") Then
    Messagebox.Show("Possible foriegn shipment.")
End If
