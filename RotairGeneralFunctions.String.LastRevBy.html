LastRevBy Method

Returns a concatenation of user and machine for change / history tracking.

Syntax
VB 
'Declarations
Public Function LastRevBy( _
) As String

Return Value

Type: System.String
The current user and machine.
Remarks

This function returns the currently logged on user and the machine name.

Example

The example sets a variable that will be used later to update a field in the history tracking.

VB 
Dim strLastrRevBy As String

'Get the last rev by information to be used later.
strLastrRevBy = Rotair_GeneralFunctions.LastRevBy()
