CurrentUserID Method

Returns the logged in employee's id.

Syntax
VB 
'Declaration
Public Function CurrentUserID( _
) As String

Return Value

Type: System.String
The employee's id.
Remarks

Returns the id of the employee who is logged in to the ERP system which can be different to the user who is logged into the computer.

Example

The example will retrieve the id of the logged in user which is found in the employee table.

VB 
Dim strID As String


strID = imsAO_Employee_CUSTOM.CurrentUserID()
