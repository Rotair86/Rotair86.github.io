ForeignCustomerScreen Method

Checks a customer for foriegn links.

Syntax
VB 
'Declarations
Public Function ForeignCustomerScreen(Of CustomerKey)( _
        CustomerID As CustomerKey _
) As Boolean

Type Parameters

CustomerKey
The type of customer key used. Can be a string or guid.

Parameters

CustomerID
Type: CustomerKey
Key for the customer.

Return Value

Type: System.Boolean
TRUE if the customer has any foreign country references.
Remarks

Returns TRUE if the customer has a foreign address, a foriegn contact or any reference to a foriegn address.

Example

The example will display a message if there is a possibility of a foriegn sale.

VB 
If ExportManagement.ForeignCustomerScreen("623454") Then
    Messagebox.Show("Possible foriegn shipment.")
End If
