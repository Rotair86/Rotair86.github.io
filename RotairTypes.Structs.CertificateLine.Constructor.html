CertificateLine Constructor

Initializes a list for serial numbers residing in this structure.

Syntax
VB 
'Declaration
Public Sub New( _
        strDummy As String _
)

Parameters

strDummy
Type: System.String
Not used.
Remarks

A structure has a default constructure to initialize all value types. This constructor can not be overridden. So if we have a reference type that we need to instantiate we need a custom constructor with a least one parameter.

Example

This example shows that the string used to initialize the structure is irrelevant.

VB 

Dim ThisCertificate As CertificateLine

ThisCertificate = New CertificateLine("Cheesecake Jellybean")
