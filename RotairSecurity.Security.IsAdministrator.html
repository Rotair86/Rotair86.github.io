IsAdministrator Method

Returns true if the logged in employee is a system administrator.

Syntax
VB 
'Declaration
Public Function IsAdministrator( _
) As Boolean

Return Value

Type: System.Boolean
TRUE if the employee is an administrator.
Remarks

Returns TRUE if the logged in user is a member of the system administrator security group.

Example

The example displays a messagebox if the current user is an administrator.

VB 

If  Rotair_Security.Security.IsAdministrator() Then
    MessageBox.Show("You are a system administrator.")
End If
