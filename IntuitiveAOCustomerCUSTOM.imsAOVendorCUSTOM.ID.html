ID Method

Returns the vendor id.

Syntax
VB 
'Declaration
Public Function ID( _
        guidVendorID As Guid _
) As String

Parameters

guidVendorID
Type: System.Guid
Record id for the vendor.

Return Value

Type: System.String
String id for the vendor.
Remarks

Returns the string id for the vendor based on the record id (guid).

Example

The example will retrieve the vendor id from the guid on the vendor card.

VB 
Dim strVendorID As String


strVendorID = imsAO_Vendor_CUSTOM.ID(MainAO.AO.GetGuid("VEN_RecordID"))
