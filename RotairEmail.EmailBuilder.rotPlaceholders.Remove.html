Remove Method

Removes a token / value pair from the collection.

Syntax
VB 
'Declaration
Public Sub Remove( _
        strName As String _
)
Remarks

This function will remove a rotPlaceholder from the rotPlaceholders collection. No action is taken if the token is not in the collection.

Example

The example will remove the token "LogInUserName" and its value from the collection if it has been set. We assume a PlaceHolders collection called mcolPH has been instantiated.

VB 

mcolPH.Remove("LogInUserName")
