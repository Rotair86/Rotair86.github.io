DisplayField Property

Information about what fields to display.

Syntax
VB 
'Declaration
Public Property DisplayField As strucDisplayField()

Remarks

Set this property to an array of strucDisplayFields that will represent the fields that are to be displayed.

Example

This example displays the sales order from the Sales Order Card. It will display a field from the master (parent) and 2 fields from the line and 2 from the delivery.

VB 
Dim frmAO As New frmDisplayAO
Dim SalesOrderFields() As strucDisplayField


ReDim SalesOrderFields(4)

With SalesOrderFields(0)
    .Field = imsAO_SalesOrder_CUSTOM.FieldNames.fSOM_SalesOrderID
    .Level = 1
    .Text = "AlphaID"
End With
With SalesOrderFields(1)
    .Field = imsAO_SalesOrderLine_CUSTOM.FieldNames.fSOI_SOLineNbr
    .Level = 2
    .Text = "Line"
End With
With SalesOrderFields(2)
    .Field = imsAO_SalesOrderLine_CUSTOM.FieldNames.flSOI_IMA_ItemID
    .Level = 2
    .Text = "Item ID"
End With
With SalesOrderFields(3)
    .Field = imsAO_SalesOrderDelivery_CUSTOM.FieldNames.fSOD_RequiredQty
    .Level = 3
    .Text = "Quantity"
End With
With SalesOrderFields(4)
    .Field = imsAO_SalesOrderDelivery_CUSTOM.FieldNames.fSOD_RequiredDate
    .Level = 3
    .Text = "Date"
End With

With frmAO
    .AO = bdrSalesOrder.AO
    .DisplayField = SalesOrderFields
    .TitlePrefix = "Sales Order"
    .ShowAO()
End With
