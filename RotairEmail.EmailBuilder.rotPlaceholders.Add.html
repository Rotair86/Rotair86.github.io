Add Method

Initializes a new instance of rotPlaceholder setting the token and value pair.

Syntax
VB 
'Declaration
Public Sub Add( _
        strName As String, _
        strValue As String _
)

Parameters

strName
Type: System.String
The token we wish to add.
strValue
Type: System.String
The cooresponding value.
Remarks

Instantiates a new Placeholder object with a token and its value.

Example

The example shows how to set up a couple of placeholders for user id and machine name. A new rotPlaceholder object is created and populated in this (Add) method.

VB 
Private Sub ProcessText() 

    Dim Placeholders As rotPlaceholders


    Placeholders = New rotPlaceholders

    ' Add login user name to placeholders collection.
    PlaceHolders.Add("LogInUserName", imsApplication.CurrentUser.LogonName)
    ' Add machine name to placeholders collection.
    PlaceHolders.Add("MachineName", Environment.MachineName)
    'Continue processing. . .

End Function
