DBType Method

Returns the form name as stored in the database. The actual form names are different due to the .Net naming conventions. The names in the database are for Access forms that were used when the phrases were created.

Syntax
VB 
'Declaration
Public Function DBType( _
) As String

Return Value

Type: System.String
The form's name.
Remarks

Returns the form's (original) name based on an enumeration.

Example

The example creates a new phrase ao for the quote card.

VB 

Dim PhraseFormDestination As Rotair_Types.Structs.PhraseForm
Dim PhraseForm As RotairAO_Phrases.rotAO_PhraseForm

PhraseFormDestination.Type = RotairTypes.Enumerations.ePhraseForm.Quote
PhraseForm = New rotAO_PhraseForm(PhraseFormDestination.DBType, True)
