NoQuoteCustomer Method

Returns TRUE if the customer is flagged as "No Quote".

Syntax
VB 
'Declaration
Public Function NoQuoteCustomer(Of CustomerKey)( _
        CustomerID As CustomerKey _
) As String

Type Parameters

CustomerKey
The type of customer key used. Can be a string or guid.

Parameters

CustomerID
Type: CustomerKey
Key for the customer.

Return Value

Type: System.Boolean
TRUE if the customer is not to be quoted.
Remarks

Returns the quotability of a customer.

Example

The example will display a message if the customer associated with the open quote can not be quoted.

VB 
Dim strCustomerID As String

strCustomerID = imsAO_Customer_CUSTOM.ID(bdrQuote.AO.GetGuid("QH_CUS_RecordID"))
If imsAO_Customer_CUSTOM.NoQuoteCustomer(strCustomerID) Then
    UI.DisplayMessage("MSG_Rotair_NoQuoteCustomer", MessageBoxIcon.Information, "No Quote Customer")
End If
