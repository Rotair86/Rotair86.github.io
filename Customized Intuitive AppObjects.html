Assemblies containing objects inheriting from vendor supplied objects whose base is imsAppObject

These are the assemblies that are derived from a vendor supplied class which is in turn derived from imsAppObject. The framework finds the customizations by the _CUSTOM suffix. These classes require the custom bit to be flipped in the table. The namespaces are named identically to the assemblies and can be considered as one since the vendor has not introduced additional namespaces.
