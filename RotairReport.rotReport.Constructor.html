rotReport Constructor

Initializes a new instance of rotReport passing in a report and either parameters for the SQL or a dataset.

Syntax

        DS As DataSet _
)

Parameters

strReportName
Type: String
Primary key in the tblReport table.
strValues()
Type: String
Array of parameters used to replace tokens in a SQL statement.
DS
Type: DataSet
Data to display in the report.
Remarks

Instantiating a new rotReport object requires a report. The formatting of the report is stored in tables. The SQL is also stroed in a table. To use this SQL just supply the parmeters. Optionally a dataset can be supplied in lieu of the parmeters.

Example

This example displays a report to give the user the option to print. The actual report uses a guid instead of a string id. The string is displayed here for clarity. Also with this particular report a couple of fields need to be set. The example shows the minimum that needs to be done.

VB 
Dim Report As rotReport

Report = New rotReport("ScreeningCustomerOrder", "623454")
With Report
    .PopulateData()
    .UpdateReportHeader()
    .CopyDataToReporter()
    .PrintPreview()
End With
