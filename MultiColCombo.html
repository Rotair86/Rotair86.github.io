Remarks

To use the Multi-column combo first you need to add it to the toolbax

  1. Right click on the toolbox and select Choose Items... you will get the following window.
  2. Select Browse and browse to MultiColCombo.dll

Now add it to the form and size it appropriately.

Following is a code sample to demo populating this control.

VB 
Dim Col As MultiColCombo.MultiColumnComboBox.ColumnCollection.Column


'Not necessary unless the combo is populated when this code is run.
MultiColumnComboBox1.ClearColumns()

'Column 1
Col = MultiColumnComboBox1.AddColumn()
With Col
    .Caption = "ID"
    .Width = 50
End With
'Column 2
Col = MultiColumnComboBox1.AddColumn()
With Col
    .Caption = "Name"
    .Width = 175
End With
'Column 3
Col = MultiColumnComboBox1.AddColumn()
With Col
    .Caption = "City"
    .Width = 100
End With
'Column 4
Col = MultiColumnComboBox1.AddColumn()
With Col
    .Caption = "ST"
    .Width = 35
End With
'Column 4
Col = MultiColumnComboBox1.AddColumn()
With Col
    .Caption = "Country"
    .Width = 100
End With

'Populate a dataset
Dim DR As DataRow
Dim DT As New DataTable
DT.Columns.Add("ID", GetType(String))
DT.Columns.Add("Name", GetType(String))
DT.Columns.Add("City", GetType(String))
DT.Columns.Add("ST", GetType(String))
DT.Columns.Add("Country", GetType(String))
DR = DT.Rows.Add
DR.Item(0) = "1"
DR.Item(1) = "Al"
DR.Item(2) = "Agustine"
DR.Item(3) = "Alabama"
DR.Item(4) = "Australia"
DR = DT.Rows.Add
DR.Item(0) = "2"
DR.Item(1) = "Betty"
DR.Item(2) = "Berlin"
DR.Item(3) = "Delaware"
DR.Item(4) = "Borneo"
DR = DT.Rows.Add
DR.Item(0) = "3"
DR.Item(1) = "Charlie"
DR.Item(2) = "Carson City"
DR.Item(3) = "Connecticut"
DR.Item(4) = "Canary Islands"

MultiColumnComboBox1.DataList = DT
MultiColumnComboBox1.DropDownWidth = 500

Notice if you run this demo code that the state column was sized for an abbreviation but a full name was used. This was done to demonstrate the ellipses that appear if the text does not fit in the column.
