bdrPurchaseOrder.Saved Event

Sends out emails about export compliance.

Syntax
VB 
'Declaration
Protected Overrides Sub bdrPurchaseOrder_Saved
Remarks

After a purchase order is saved calls ForeignVendorScreen and sends out an email if:

Example

Calls the base then sends out an email if the vendor has any foreign contacts and a defense article has been added.

VB 
Protected Overrides Sub bdrPurchaseOrder_Saved( _
    ByVal semder As System.Object, _
    ByVal e As System.EventArgs _
    )

    Dim strPurchaseOrderID As String
    Dim guidVendorRecordID As Guid


    MyBase.bdrPurchaseOrder_Saved(sender, e)

    'Get the purchase order id
    strPurchaseOrderID = bdrPurchaseOrder.AO.GetString(AOPurchaseOrderWrapper.FieldNames.fPOM_PurchOrderID)
    'Get the vendor's record id
    guidVendorRecordID = New Guid(bdrPurchaseOrder.AO.GetString(AOPurchaseOrderWrapper.FieldNames.fPOM_VEN_RecordID))

    'Check for a foreign vendor
    If ExportManagement.ForeignVendorScreen(guidVendorRecordID) Then
        'Check for any defemse articles
        If CheckForNewlyAddedDefenseArticles() Then
            'Is this a new purchase order?
            If bdrPurchaseOrder.AO.DocumentState = IntuitiveAppObjectFramework2.aoObjectState.NEW_DOCUMENT Then
                E_Mail.rotEmailBuilder.SendPurchaseOrderEMail( _
                    Rotair_Types.Enumerations.eEmailForPurchaseOrder.NewPOWithForeignVendorAndDefenseArticle, _
                    guidVendorRecordID, strPurchaseOrderID)
            Else
                E_Mail.rotEmailBuilder.SendPurchaseOrderEMail( _
                    Rotair_Types.Enumerations.eEmailForPurchaseOrder.ExistingPOWithForeignVendorAddedDefenseArticle, _
                    guidVendorRecordID, strPurchaseOrderID)
            End If
        End If
    End If

End Sub
