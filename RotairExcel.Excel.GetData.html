GetData Method

Converts a spreadsheet to a dataset.

Syntax
VB 
'Declaration
Public Sub GetData( _
) As DataSet

Return Value

Type: System.DataSet
Represents the data from the spreadsheet.
Remarks

This will get the data from the spreradsheet into a dataset. Was to be used with the supplier / manufacturer form that may be discontinued.

Example

This example creates a new dataset and populates it with the spreadsheet after populating the spreadsheet from a grid..

VB 
Dim DS As DataSet

XL = New rotExcel("C:\My Documents\Spreadsheet.xls")
XL.ShowData(Enumerations.eExcelReport.Booking, Grid.DataSource)
DS = XL.GetData()
