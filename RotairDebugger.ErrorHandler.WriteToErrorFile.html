WriteToErrorFile Method

Used to store debugging information to a file.

Syntax
VB 
'Declaration
Public Shared Function WriteToErrorFile( _
    strModule As String,
    strMethod As String,
    strErrorMessage As String
    ) As Boolean

Parameters

strModule
Type: System.String
The class where the error occured.
strMethod
Type: System.String
The function which called this method.
strErrorMessage
Type: System.String
Text of the error message.

Return Value

Type: System.Boolean
The status of the attempt to write out the passed in information.
Remarks

Writes out error information to a file. Uses reflection to get the directory where the code was launched from.

Example

The example shows how to put the call in a Try . . . Catch block to write out error information. It returns TRUE if successful.

VB 
Private Function DoSomething() As Boolean

    Dim rc As Boolean


    Try

        'Do some processing.
        'Process
        'Process
        'Process
        rc = True

    Catch ex As Exception
        rc = False
        Debugger.ErrorHandler.WriteToErrorFile( _
            ex.TargetSite.ReflectedType.Name, _
            ex.TargetSite.Name, _
            ex.Message)

    End Try

    Return rc

End Function
