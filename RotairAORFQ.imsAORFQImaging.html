Remarks

This class represents a response from a vendor. It also represents a file on a network share and a record in either tblRFQImaging or tblRFQVendorQuote.

The Delete method should only be called when an error is made when indexing.

The iterator is just a counter used as a key.

The records in the legacy table (tblRFQImaging) do not allow as many lookup options as the newer structure. So some lookup results can miss some older records if filtered by either Item or Vendor. Looking up by RFQ ID should always work.
