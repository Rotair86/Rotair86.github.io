ShowStatusButtons Method

Displays buttons on the footer panel located at the bottom of imsForms.

Syntax
VB 
'Declaration
Public Function ShowStatusButtons( _
        pnl As imsPanel, _
        eWhich As eShowStatus _
) As Boolean

Parameters

pnl
Type: imsPanel
The panel to add buttons to.
eWhich
Type: eShowStatus
The type of button to add.

Return Value

Type: System.Boolean
TRUE if no problems were encountered.
Remarks

Adds a button in the first location for either No Quote or DP Stop OR adds a button in the second location to indicate Notes. Handlers must be added in the UI in order to get the buttons to do anything.

Example

This example will light up the "DP Stop" button if the customer associated with the current sales order is a denied party.

VB 
If SalesOrder.ScreenResults = eScreeningRC.Denied_Party Then
    UI.ShowStatusButtons(pnlFooter, UI.eShowStatus.Denied)
End If
