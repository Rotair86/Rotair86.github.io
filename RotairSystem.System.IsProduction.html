IsProduction Method

Returns true if connected to the production database.

Syntax
VB 
'Declaration
Public Function IsProduction( _
) As Boolean

Return Value

Type: System.Boolean
TRUE if connected top the production database.
Remarks

Returns TRUE if the database name does not contain an underscore ("_"). The development and test databases have a suffix with an underscore, production does not.

Example

The example displays a messagebox if we are in production.

VB 

If  Rotair_System.System.IsProduction() Then
    MessageBox.Show("We are in production.")
End If
