NameByID Method

Returns the employee's name.

Syntax
VB 
'Declaration
Public Function NameByID(Of EmployeeKey)( _
        eNameStyle As eNameReturnStyle, _
        UserID As EmployeeKey _
) As String

Type Parameters

EmployeeKey
The type of employee key used. Can be a string or guid.

Parameters

eNameStyle
Type: eNameReturnStyle
Determines the format that the name is returned.
UserID
Type: EmployeeKey
Key for the employee.

Return Value

Type: System.String
The employee's name in the requested format.
Remarks

Returns the employee's name by either the employee id or record id in the desired format (style).

Example

The example will retrieve the employee name two different ways and in two different styles.

VB 
Dim guidEmployeeID As Guid
Dim strName As String


guidEmployeeID = New Guid("9867C13A-D208-49E6-B7E9-71E5E0245773")

'Returns "Maher, John"
strName = imsAO_Employee_CUSTOM.NameByID(eNameReturnStyle.LastFirst, "943")

'Returns "943 - John Maher -"
strName = imsAO_Employee_CUSTOM.NameByID(eNameReturnStyle.IDFirstLast, guidEmployeeID)
