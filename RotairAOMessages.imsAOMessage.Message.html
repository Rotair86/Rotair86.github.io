Message Method

Returns a message.

Syntax
VB 
'Declaration
Public Function Message( _
        strMessageID As String _
) As String

Parameters

strMessageID
Type: System.String
The message id.

Return Value

Type: System.Data.DataSet
The message text.
Remarks

Returns a message based on an id.

Example

The example will display a screening question by looking up the question with the id. It also uses a custom message box display function.

VB 

Dim dr As DialogResult


dr = CustomMessage.DisplayScreeningFailure(SalesOrder.ScreenResults, _
        SalesOrder.ScreenAction, _
        imsAO_Message.Message("MSG_Rotair_PartyScreeningWanna"), _
        New String() {"002436"}, _
        MessageBoxButtons.YesNoCancel)
