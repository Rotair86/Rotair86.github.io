LastError Property

Provides access to error text.

Syntax
VB 
'Declarations
Public Shared Property LastError As String

Remarks

Holds the last error that was trapped. An error can be stored here that happened in a much earlier function. So it is recommended to check this property to ensure it is clear before calling any functions where an error may be expected.

Example

The example will try to get the computer name. If the LastError property is empty before the call but contains something after the call then we know an error occurred. If it is not empty before the call then we can't be sure. If it is empty before and after then we know the call was successful.

VB 
Dim strComputerName As String
Dim bErrorClear As Boolean


If Rotair_GeneralFunctions.LastError = String.Empty Then
    bErrorClear = True
End If

Rotair_GeneralFunctions.ComputerName(strComputerName)
If Rotair_GeneralFunctions.LastError <> String.Empty Then
    If bErrorClear Then
        MessageBox.Show("Error occured:" & Rotair_GeneralFunctions.LastError)
    Else
        MessageBox.Show("Error may have occured:" & Rotair_GeneralFunctions.LastError)
    End If
Else
    MessageBox.Show("Computer name is:" & strComputerName)
End If
