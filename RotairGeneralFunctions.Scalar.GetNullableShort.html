GetNullableShort Method

Returns a value supplying a default for nothing.

Syntax
VB 
'Declarations
Public Function GetNullableShort( _
        value As Short? _
) As Short

Parameters

value
Type: Nullable
A short that can be nothing.

Return Value

Type: System.Short
The passed in value or 0 if nothing was passed in.
Remarks

Pass this function a nullable short and it will return the short value if it is not nothing. If the passed in value is nothing then it will return 0.

Example

The example will set a variable. But we may not have data, that is why the variable is defined as nullable. So we call the function that will assign a default value if nothing was assigned to the variable.

VB 
Dim shtSomeShort As Short?
Dim shtAnotherShort As Short


'Let's load the short shtSomeShort.
LoadData(shtSomeShort)
'Now we need to convert shtSomeShort (which may or may not have been set) to shtAnotherShort.
shtAnotherShort = Rotair_GeneralFunctions.GetNullableShort(shtSomeShort)
