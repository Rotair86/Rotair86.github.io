LoadPurchaseOrder Function

Loads the notes associated with a purchase order.

Syntax
VB 
'Declaration
Protected Overrides Sub LoadPurchaseOrder
Remarks

Loads the notes after the purchase order. Refreshes the grid views to ensure the unbound columns get displayed. Also will display a label in the footer if notes exist.

Example

Fills the notes collections then checks to see if we need to light up the "Notes Exist" label.

VB 
Protected Overrides Sub LoadPurchaseOrder()

    mcolNotes = New rotAO_NoteType.rotAO_Note_Collection

    mcolNotes.NotesForParentKeyByTypeWithChildNotes(New Guid( _
        bdrPurchaseOrder.AO.GetString("POM_RecordID")), _
        rotAO_NoteType.eNoteType.PurchaseOrder.ToString)

    If mcolNotes.Count > 0 Then
        RotairUI_Common.UI.ShowStatus(pnlFooter, RotairUI_Common.UI.eShowStatus.NotesExist)
    End If

End Sub
