SendDeniedFlagEMail Method

Sends an e-mail when a customer or vendor gets "denied".

Syntax
VB 
'Declaration
Public Shared Function SendDeniedFlagEMail(Of PartyKey)( _
        ePartyType As ePartyType, _
        PartyID As PartyKey, _
        strEmailTypeID As String _
) As Boolean

Type Parameters

PartyKey
The type of key used. Can be a string or guid.

Parameters

ePartyType
Type: ePartyType
The specific type which can be customer or vendor.
PartyID
Type: PartyKey
Key for the customer or vendor.
strEmailTypeID
Type: System.String
The specific type of mail, primary key on tblEmailType.

Return Value

Type: System.Boolean
TRUE if no problems were found.
Remarks

Sets all the placeholders needed for setting the denied flag mail then calls the SendClassBasedEmail function.

Example

The example assumes we have already populated the various variables. If we did not get an e-mail address then an e-mail is sent to the administrator, otherwise the e-mail destined for the employee goes out.

VB 
'Send an e-mail when marking a customer as a denied party.
If SendDeniedFlagEMail(Enumerations.ePartyType.Customer, strCustomerID, "DPSetCustomer") Then
    'Success.
    MessageBox.Show("E-mail sent.")
Else
    'Possible failure.
    MessageBox.Show("E-mail indicating a customer was denied may not have went out.")
End If
