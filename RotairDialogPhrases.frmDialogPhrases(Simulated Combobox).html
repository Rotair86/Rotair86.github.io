Simulated Combobox

This form uses a simulated combobox. The original form in Access used a combobox. Users reported that it would help them to pick more than one phrase at a time. A combobox in any form does not allow multi-selecting. To retain the original look and feel at close as possible and to include new multi- select capability we simulate a combobox with a listbox, actually two of them, and a button.

Remarks

This form can function in two modes. Single-select and multi-select. The height is slightly affected because items with a checkbox (multi-select) are slightly taller than a similiar item without the checkbox (single select).

Both modes present the user with a listbox compressed to a single row, although it is a checkedlistbox in multi-select. A button with a downward pointing chevron is beside the listbox that will expand it (if the listbox is a single line in height) or shrink it (when iit is expanded) when clicked. That is where the similarities end.

Single Select

The dropped down list behaves as if it were a combobox with the DropDownStyle set to DropDownList. Once an item is selected it is added to the textbox and the listbox is shrunk to one line. To shrink the listbox without adding a phrase the user must click the chevron.

Multi-Select

The dropped down list allows checking and unchecking of items. Multi-select mode is easily recognizable by the checkboxes to the left of items in the list. When the user is satisified that they have made all the selections that they wish they can click the Select button to add the phrases to the textbox. They can also double-click the dropped down list to select the choices, but it can be confusing because the control interprets a double click as two single clicks also (checking and unchecking an item).
