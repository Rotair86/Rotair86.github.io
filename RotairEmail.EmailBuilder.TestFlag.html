TestFlag Property

Turns on a messagebox to display mails.

Syntax
VB 
'Declarations
Public Shared Property TestFlag As Boolean

Remarks

If this property is TRUE then any e-mails will be displayed in a message box instead of going to the recipients.

Example

The example tests the SendScreeningEMail function.

VB 
'Set the flag.
Rotair_Email.rotEmailBuilder.TestFlag = True
'Test the results.
SendScreeningEMail(Enumerations.ePartyType.Customer, _
            "000000", _
            Enumerations.eScreenAction.Open_A_Customer, _
            "TestForm")
'Reset the flag.
Rotair_Email.rotEmailBuilder.TestFlag = False
