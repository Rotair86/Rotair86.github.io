FieldHasChanged Method

Returns TRUE if the passed in field has had its value changed.

Syntax
VB 
'Declaration
Public Function FieldHasChanged( _
        AO As imsAppObject, _
        Field As String _
) As Boolean

Parameters

AO
Type: imsAppObject
Not passed in by the caller. This identifies what object the function is applicable to.
Field
Type: System.String
The field to check for changes.

Return Value

Type: System.Boolean
True if the value represented by the field is not the same as the value originally set.
Remarks

Allows for cleaner code when using any object that derives from imsAppObject.

Example

The example displays a messagebox if the customer was changed.

VB 

Dim aoSalesOrder As imsAO_SalesOrder

If aoSalesOrder.FieldHasChanged("SOM_CUS_RecordID") Then
    Messagebox.Show("Customer has been changed")
End If
