CantAddNotes Method

Displays a message.

Syntax
VB 
'Declaration
Public Sub CantAddNotes( _
)
Remarks

Displays a message when notes can not be added because the parent has not been created yet.

Example

This example shows how this property is used. Since a note needs a parent id and a parent id isn't created until the underlying object is created a note can't be created until a grid row is created which will in turn create the underlying object (which will create its id). This means that a note can be added anytime EXCEPT as the first field. So when the Notes button is pressed we check the rowhandle. If the row has not been created, we get a constant that matches the InvalidHandle property and we call the CantAddNotes function to display a message instead of creating the note.

VB 
Private Sub GridView_Notes_ButtonPressed( _
            ByVal sender As Object,
            ByVal e As DevExpress.XtraEditors.Controls.ButtonPressedEventArgs) _
            Handles mNotesButton.ButtonPressed

    If GridView.FocusedRowHandle <> UI.InvalidHandle Then
        'Process the Note
    Else
        UI.CantAddNotes()
    End If

End Sub
