rotPlaceholder Constructor

Initializes a new instance of rotPlaceholder setting the token and value pair.

Syntax
VB 
'Declaration
Public Sub New( _
        strName As String, _
        strValue As String _
)
Remarks

Instantiating a new Placeholder object requires a token and its value. This object is pointless without the pair. It is for this reason that the parameterless constructor has been marked private.

Example

This example creates a new placeholder object setting the token and corresponding value.

VB 
Dim PH As rotPlaceholder


PH = New rotPlaceholder("Token", "Value")
