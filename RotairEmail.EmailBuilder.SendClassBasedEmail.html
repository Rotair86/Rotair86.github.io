SendClassBasedEmail Method

Sends out e-mail based on predefined messages and recipient lists.

Syntax
VB 
'Declaration
Public Shared Sub SendClassBasedEmail( _
        strMessageType As String, _
        PlaceHolders As rotPlaceHolders _
) As Boolean

Public Shared Sub SendClassBasedEmail( _
        strMessageType As String, _
        PlaceHolders As rotPlaceHolders, _
        strUserIDTest As String _
) As Boolean

Public Shared Sub SendClassBasedEmail( _
        strMessageType As String, _
        PlaceHolders As rotPlaceHolders, _
        bMessageBoxTest As Boolean _
) As Boolean

Parameters

strMessageType
Type: System.String
Message type is a key used to get a specific e-mail.
PlaceHolders
Type: rotPlaceholders
Collection of token / value pairs used in the e-mail.
strUserIDTest
Type: System.String
If this parameter is not null then the recipient list is ignored and the value in this variable is used as the recipient.
bMessageBoxTest
Type: System.Boolean
If TRUE then the mail goes to a message box instead.

Return Value

Type: System.Boolean
TRUE if no problems were found.
Remarks

Sets all the placeholders needed for administrator mail then calls the SendClassBasedEmail function.

Note: in order to find out that an e-mail address is missing the function SendClassBasedEmail would have already been called once. If the administrator's e-mail address is not in the database we would end up with a loop untill we got a stack overflow. The flag mbSendAdministratorEMail ensures that this down not happen.

Example

The example assumes we have already popualted the various variables. If we did not get an e-mail address then an e-mail is sent to the administrator, otherwise the e-mail destined for the employee goes out.

VB 
Private Function SendEmailForNewSalesOrder() As Boolean

    Dim Placeholders As rotPlaceholders
    Dim rc As Boolean


    ' Instantiate placeholders.
    Placeholders = New rotPlaceholders

    ' Add login user name to placeholders collection.
    PlaceHolders.Add("LogInUserName", imsApplication.CurrentUser.LogonName)
    ' Add machine name to placeholders collection.
    PlaceHolders.Add("MachineName", Environment.MachineName)

    'Send the email.
    rc = SendClassBasedEmail("NewSalesOrder", Placeholders)
    If rc Then
        'Success.
        MessageBox.Show("Email sent.")
    Else
        'Possible failure.
        MessageBox.Show("Email may not have gone out.")
    End If

    Return rc

End Function
