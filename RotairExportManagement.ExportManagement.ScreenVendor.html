ScreenVendor Method

Checks a vendor's screening.

Syntax
VB 
'Declarations
Public Function ScreenVendor(Of VendorKey)( _
        VendorID As VendorKey, _
        eSourceAction As eScreenAction, _
        VendorScreening As rotAO_Screening _
) As eScreeningRC

Type Parameters

VendorKey
The type of vendor key used. Can be a string or guid.

Parameters

VendorID
Type: VendorKey
Key for the vendor.
eSourceAction
Type: eScreenAction
What the user did to instigate the screening.
VendorScreening
Type: rotAO_Screening
A screening record for the vendor.

Return Value

Type: eScreeningRC
Results of the screening.
Remarks

Returns the results of a vendor screening.

Example

The example will display a message depending on the results of the vendor's screening.

VB 
eRC As Enumerations.eScreeningRC
Screening As rotAOScreening

eRC = ExportManagement.ScreenVendor( _
        "176930", _
        Enumerations.eScreenAction.Open_A_Vendor, _
        Screening)
Select Case eRC
    Case Enumerations.eScreeningRC.Denied_Party
        Messagebox.Show("Can not do business with this vendor.")
    Case Enumerations.eScreeningRC.Never_Screened 
        Messagebox.Show("Vendor needs to be screened.")
    Case Enumerations.eScreeningRC.Screening_Out_of_Date
        Messagebox.Show("Vendor needs a current screening.")
    Case Enumerations.eScreeningRC.Screened_OK
        Messagebox.Show("Clear to do business with this vendor.")
End Select
