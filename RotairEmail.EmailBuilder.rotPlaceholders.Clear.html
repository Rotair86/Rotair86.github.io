Clear Method

Empties the collection of members.

Syntax
VB 
'Declaration
Public Sub Clear
Remarks

Disposes of all the objects in the collection freeing up the memory.

Example

Populates the collection, does some processing then resets the collection.

VB 
Dim colPH As New rotPlaceholders

'Add some tokens.
colPH.Add("FirstName", "John")
colPH.Add("LastName", "Maher")

'Do some processing.
If ProcessData(colPH) Then
    'Clear the tokens out of the collection if processed successfully.
    colPH.Clear
End If
