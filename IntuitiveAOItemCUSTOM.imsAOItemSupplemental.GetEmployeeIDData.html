GetEmployeeIDData Function

Sets a field to the employee id of the currently logged in user.

Syntax
VB 
'Declaration
Protected Overrides Sub GetEmployeeIDData()
Remarks

This function gets called by the framework.

Example

This function calls the SetEmployeeIDData function passing in the field which is to hold the employee id. This is how intuitive recommended to do this.

VB 
Protected Overrides Sub GetEmployeeIDData()

    'Sets the field LastRevBy to the currently logged in employee.
    SetEmployeeIDData(fLastRevBy, False)
    'May be equivalent to:
    '    SetValue(True, fLastRevBy, imsApplication.CurrentUser.EmpID)

End Function
