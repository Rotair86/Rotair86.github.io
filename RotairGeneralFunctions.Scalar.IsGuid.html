IsGuid Method

Returns TRUE if the passed in string can be treated as a guid.

Syntax
VB 
'Declarations
Public Function IsGuid( _
        strPossibleGuid As String _
) As Boolean

Parameters

strPossibleGuid
Type: System.String
A string to check to see if it can be converted to a guid.

Return Value

Type: System.Boolean
TRUE if the string will convert to a guid.
Remarks

Pass this function a string and it will return TRUE if it can be converted to a guid.

Example

The example will get some text that may be a guid. It then checks, and if it passes the test converts it to a guid.

VB 
Dim strSomePossibleGuid As String
Dim guidSomeGuid As Guid


'Let's get some text that may be a guid.
strSomePossibleGuid = GetData()
'Now we need to check to see if we have a valid guid.
If Rotair_GeneralFunctions.IsGuid(strSomePossibleGuid)Then
    guidSomeGuid = New Guid(strSomePossibleGuid)
End If
