eExcelReport Enumeration

Types of reports that can have their data transferred to Excel. There may be other reports that can go to Excel, but these either need some special formatting to look good or a special selection dialog or both.

Members
  Member name Description
None Default if no value is assigned.
Backlog Report generated by a selection dialog of Sales Orders logged (booked).
Booking Report generated by a selection dialog of Sales Orders logged (booked).
ManufacturedMaterialsToInspect Parameterless report. When it is run is what will determine the dataset of manufactured items to be inspected.
PurchasedMaterialsToInspect Parameterless report. When it is run is what will determine the dataset of purchased material to be inspected.
SalesOrder Report generated by a selection dialog used to see the material type.
Shipments A report on shipments and values that would filter on Customer Territory codes.
StockValuePurchased Parameterless report. When it is run is what will determine the dataset of purchased inventory with a value based on the accounting value times the quantity.
StockValueManufactured Parameterless report. When it is run is what will determine the dataset of manufactured inventory with a value based on the accounting value times the quantity.
Wip Parameterless report. When it is run is what will determine the dataset of Work Orders in process valuation.
