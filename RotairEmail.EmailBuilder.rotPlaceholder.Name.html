Name Property

Provides access to a token placeholder.

Syntax
VB 
'Declarations
Public Property Name As String

Remarks

This property returns the token which is replaced by its partner, Value, in the text where the token is found.

Example

The example will return TRUE if there is a Zip Code token in the collection.

VB 
Private Function CheckForZip() As Boolean

    Dim bFound As Boolean
    Dim PH As rotPlaceholder


    For Each PH In mcolPH
        If PH.Name = "ZipCode" Then
            bFound = True
            Exit For
        End If
    Next

    Return bFound

End Function
