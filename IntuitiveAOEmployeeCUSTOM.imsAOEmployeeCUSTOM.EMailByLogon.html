EMailByLogon Method

Returns the e-mail address of the logged in user.

Syntax
VB 
'Declaration
Public Function EMailByLogon( _
        strUserLogon As String _
) As String

Parameters

strUserLogon
Type: System.String
The employee's logon id.

Return Value

Type: System.String
The employee's e-mail address.
Remarks

Returns the e-mail address of the employee who is logged in to the ERP system which can be different to the user who is logged into the computer.

Example

The example will retrieve the e-mail address of the user whoose logon id is passed in.

VB 
Dim strEMail As String


'Returns "JohnM@rotair.com")
strEMail = imsAO_Employee_CUSTOM.EMailByLogon("JohnM")
