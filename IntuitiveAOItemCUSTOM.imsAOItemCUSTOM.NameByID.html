NameByID Method

Returns an item name.

Shared function because we do not need an object for the results and also to self document.

Syntax
VB 
'Declaration
Public Shared Function NameByID(Of ItemKey)( _
        ItemID As ItemKey _
    ) As DataSet

Type Parameters

ItemKey
The type of item key used. Can be a string or guid.

Parameters

ItemID
Type: ItemKey
Key for the item.

Return Value

Type: System.String
The item name.
Remarks

Returns the item name based on the id.

Example

The example retrieves the item name with two different ids.

VB 
Dim strID As String
Dim strName As String
Dim guidRecordID As Guid


strID = "Widget"
guidRecordID = New Guid("DEADBEEF-BAAD-F00D-CAFE-BA5EBA11BABE")

strName = imsAO_Item_CUSTOM.NameByID(strID)
strName = imsAO_Item_CUSTOM.NameByID(guidRecordID)
