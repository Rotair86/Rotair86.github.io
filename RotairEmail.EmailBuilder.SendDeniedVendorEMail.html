SendDeniedVendorEMail Method

Sends an e-mail when a vendor gets "denied" if they have open purchase orders or RFQs.

Syntax
VB 
'Declaration
Public Shared Function SendDeniedVendorEMail( _
        guidVendorRecordID As guid, _
        strPOCount As String, _
        strRFQCount As String, _
        strPOList As String, _
        strRFQList As String _
) As Boolean

Parameters

guidVendorRecordID
Type: System.Guid
The record id for a vendor. Not a primary key, but a unique value that can be used as one.
strPOCount
Type: System.String
The number of open purchase orders for a specific vendor.
strRFQCount
Type: System.String
The number of open RFQs for a specific vendor.
strPOList
Type: System.String
The open purchase order list.
strRFQList
Type: System.String
The open RFQ list.

Return Value

Type: System.Boolean
TRUE if no problems were found.
Remarks

Sets all the placeholders needed building the string of open purchase orders and RFQs including a list of them along with the count then calls the SendClassBasedEmail function.

Example

The example send out an e-mail indication the a vendor has been denied. Then retrieves the number of open purchase orders and open RFQs. If there are any then sends out an e-mail containing this information.

VB 
Private Function SendEmailForDeniedVendor(ByVal guidVendorRecordID As Guid) As Boolean

    Dim intPOCount As Integer
    Dim strPOList As String
    Dim intRFQCount As Integer
    Dim strRFQList As String
    Dim rc As Boolean


    'Send an email for denying the vendor.
    rc = SendDeniedFlagEMail(Rotair_Types.Enumerations.ePartyType.Vendor, guidVendorRecordID, "DPSetVendor")

    'Get the PO and RFQ counts and lists.
    GetOpenPOsAndRFQs(intPOCount, strPOList, intRFQCount, strRFQList)

    'Check to see if we need to send an email.
    If intPOCount + intRFQCount > 0 Then
        'Send the mail.
        rc = rc Or SendDeniedVendorEMail(guidVendorRecordID, intPOCount.ToString, intRFQCount.ToString, strPOList, strRFQList)
    End If

    Return rc

End Function
