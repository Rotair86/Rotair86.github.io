ComputerName Method

Returns the computer name.

Syntax
VB 
'Declarations
Public Function ComputerName( _
        strComputerName As String _
) As Boolean

Parameters

strComputerName
Type: System.String
The computer name.

Return Value

Type: System.Boolean
TRUE if no errors occured.
Remarks

Stuffs the passed in variable with the computer name.

Example

The example retrieve the computer name and display it in a message box.

VB 
Dim strComputerName As String


Rotair_GeneralFunctions.ComputerName(strComputerName)
MessageBox.Show("Computer name is:" & strComputerName)
