Exists Method

Checks for the existence of a token.

Syntax
VB 
'Declaration
Public Function Exists( _
        strName As String _
)  As Boolean

Parameters

strName
Type: System.String
The token whose existance we wish to check.

Return Value

Type: System.Boolean
TRUE if the token exists in the collection.
Remarks

Pass this function a name of a token and it returns TRUE if the token exists in the collection.

Example

The example will display a message box if the token "LogInUserName" is found in the collection.

VB 
If mcolPH.Exists("LogInUserName")
    MessageBox.Show("LogInUserName found in the collection.")
End If
