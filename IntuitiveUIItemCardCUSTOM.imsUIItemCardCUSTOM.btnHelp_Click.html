btnHelp_Click Event

Displays the ao form if a flag is set.

Syntax
VB 
'Declaration
Private Event Click As EventHandler
Remarks

If mbControlShift is TRUE then will display the ao form. This event is handled instead of overridding the base implementation. This is because the button is serving a dual purpose and the functionality is best isolated.

Example

Displays the ao form depending on a flag.

VB 
Private Sub btnHelp_Click( _ 
    ByVal semder As System.Object, _
    ByVal e As System.EventArgs _
    ) Handles btnHelp.Click

    If mbControlShift Then
        DisplayItem("Item")
    End If

End Sub
